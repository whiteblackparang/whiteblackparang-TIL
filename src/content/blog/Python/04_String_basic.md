# 문자열 기초 String Basic 

## 문자열 연결하기 
- 문자열은 + 연산자로 연결
- 두 개 이상의 문자열을 하나로 합칠 때 사용
- 문자열에서는 연결의 의미
- 공백을 넣으려면 ' ' 처럼 공백 문자열을 중간에 추가


## 문자열 반복하기
- 연산자를 사용하면 문자열을 반복 가능
- 같은 문자를 여러 번 출력할 때 유용

## 문자열 길이
- len 함수는 문자열의 길이를 반환
- 문자열에 포함된 문자의 개수 셈 
- 공백 & 특수문자 모두 포함

## f-string 포맷팅 
- f-string은 문자열 앞에 f를 붙여 변수를 직접 넣을 수 있는 방법
- + 연결이나 str 변환 없이도 변수와 문자를 조합
- 중괄호 {} 안에 변수명이나 표현식을 넣으면 자동으로 문자열로 변환

## 줄바꿈 문자
- 이스케이프 문자는 백슬래시\로 시작하는 특수 문자
- \n은 줄바꿈
- 문자열 중간에 사용하면 그 지점에서 줄이 바뀜
- 여러 줄 출력에 유용

## 탭문자 
- \t는 탭 문자
- 일정한 간격을 만든다
- 텍스트를 정렬할 때 유용

## 백슬래시 문자
- 백슬래시 자체를 문자로 표현하려면 \\처럼 두 번 사용

## 여러 줄 문자열 
- 삼중 따옴표''' 또는 \"\"\"를 사용
- 줄바꿈이 자동으로 포함
- \n을 사용하지 않아도 됨
