# Dictionary Methods

딕셔너리 메서드는 딕셔너리의 데이터를 조회, 수정, 삭제하거나 새로운 데이터를 추가할 때 사용하는 기능이다.

---

## get()

- 키(Key)의 값을 안전하게 가져온다.
- 키가 존재하면 해당 값을 반환한다.
- 키가 없으면 `None`을 반환한다.
- 기본값을 지정할 수도 있다.
- 설정값이나 옵션 처리에서 자주 사용된다.

---

## keys()

- 딕셔너리의 모든 키를 반환한다.
- 반환형은 `dict_keys` 객체이다.
- `list()`로 변환하면 리스트처럼 사용할 수 있다.
- 반복문에서 키를 순회할 때 자주 사용한다.

---

## values()

- 딕셔너리의 모든 값을 반환한다.
- 반환형은 `dict_values` 객체이다.
- `list()`로 변환하여 사용할 수 있다.
- 값의 합계나 평균을 계산할 때 활용된다.

---

## items()

- 키와 값을 함께 반환한다.
- `(key, value)` 형태의 튜플을 반환한다.
- 반복문에서 가장 많이 사용하는 메서드이다.

---

## update()

- 다른 딕셔너리의 내용을 현재 딕셔너리에 병합한다.
- 같은 키가 있으면 기존 값을 덮어쓴다.
- 새로운 키는 자동으로 추가된다.
- 설정 정보나 사용자 데이터를 합칠 때 자주 사용한다.

---

## pop()

- 지정한 키를 삭제하고 값을 반환한다.
- 키가 없으면 `KeyError`가 발생한다.
- 기본값을 함께 지정하면 에러 없이 사용할 수 있다.

---

## popitem()

- 마지막 키-값 쌍을 삭제하고 반환한다.
- 반환값은 `(key, value)` 튜플이다.
- 빈 딕셔너리에서 사용하면 `KeyError`가 발생한다.

---

## clear()

- 딕셔너리의 모든 데이터를 삭제한다.
- 빈 딕셔너리(`{}`)가 된다.
- 반환값은 `None`이다.

---

## setdefault()

- 키가 존재하면 기존 값을 반환한다.
- 키가 없으면 기본값을 추가한 뒤 반환한다.
- 딕셔너리 초기화나 그룹화 작업에서 자주 사용된다.

---

# 자주 사용하는 Dictionary 메서드

| 메서드 | 설명 |
|---------|------|
| `get()` | 안전하게 값 조회 |
| `keys()` | 모든 키 반환 |
| `values()` | 모든 값 반환 |
| `items()` | 키와 값 반환 |
| `update()` | 병합 및 수정 |
| `pop()` | 특정 키 삭제 |
| `popitem()` | 마지막 항목 삭제 |
| `clear()` | 모든 데이터 삭제 |
| `setdefault()` | 키가 없으면 기본값 추가 |

---

# 핵심 정리

- `get()`은 키가 없어도 에러가 발생하지 않는다.
- `keys()`, `values()`, `items()`는 딕셔너리를 순회할 때 자주 사용한다.
- `update()`는 데이터를 병합하거나 수정할 때 사용한다.
- `pop()`과 `popitem()`은 데이터를 삭제하면서 값을 반환한다.
- `clear()`는 모든 데이터를 삭제한다.
- `setdefault()`는 키가 없을 때 기본값을 추가한다.
