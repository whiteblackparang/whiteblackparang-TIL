# 조건문이란?

조건문(Conditional Statement)은 **특정 조건의 참(True) 또는 거짓(False)에 따라 서로 다른 코드를 실행하는 기능**이다.

프로그램은 항상 위에서 아래로 순차적으로 실행된다. 하지만 모든 상황에서 같은 동작을 수행하는 것은 아니다.

예를 들어 다음과 같은 경우에는 조건문이 필요하다.

- 시험 점수가 90점 이상이면 A를 출력한다.
- 나이가 19세 이상이면 성인으로 판단한다.
- 구매 금액이 10만 원 이상이면 할인한다.
- 결측치(NA)가 있으면 제거한다.

이처럼 **조건에 따라 프로그램의 흐름을 제어하는 것이 조건문의 역할**이다.

---

# if 문

## 문법

```r
if(조건){

    실행 코드

}
```

조건이 TRUE이면 중괄호 안의 코드가 실행된다.

---

## 예제 1

```r
score <- 95

if(score >= 90){

    print("A등급")

}
```

결과

```
[1] "A등급"
```

---

## 예제 2

```r
age <- 25

if(age >= 20){

    print("성인입니다.")

}
```

결과

```
[1] "성인입니다."
```

조건이 FALSE이면 아무것도 실행되지 않는다.

---

# if ~ else 문

조건이 참이면 첫 번째 코드를 실행하고, 거짓이면 else 부분을 실행한다.

## 문법

```r
if(조건){

    실행 코드

}else{

    실행 코드

}
```

---

## 예제 1

```r
score <- 75

if(score >= 80){

    print("합격")

}else{

    print("불합격")

}
```

결과

```
[1] "불합격"
```

---

## 예제 2

```r
number <- 8

if(number %% 2 == 0){

    print("짝수")

}else{

    print("홀수")

}
```

결과

```
[1] "짝수"
```

`%%`는 나머지를 구하는 연산자이다.

---

# if ~ else if ~ else 문

조건이 여러 개일 때 사용한다.

## 문법

```r
if(조건1){

    실행 코드

}else if(조건2){

    실행 코드

}else{

    실행 코드

}
```

---

## 예제 1. 학점 계산

```r
score <- 87

if(score >= 90){

    print("A")

}else if(score >= 80){

    print("B")

}else if(score >= 70){

    print("C")

}else{

    print("F")

}
```

결과

```
[1] "B"
```

---

## 예제 2. BMI 판정

```r
bmi <- 24

if(bmi < 18.5){

    print("저체중")

}else if(bmi < 23){

    print("정상")

}else if(bmi < 25){

    print("과체중")

}else{

    print("비만")

}
```

결과

```
[1] "과체중"
```

# 비교 연산자

조건문에서는 비교 연산자를 자주 사용한다.

| 연산자 | 의미 |
|--------|------|
| == | 같다 |
| != | 같지 않다 |
| > | 크다 |
| < | 작다 |
| >= | 크거나 같다 |
| <= | 작거나 같다 |

예시

```r
5 > 3
```

결과

```
TRUE
```

---

# 논리 연산자

여러 조건을 함께 사용할 수도 있다.

| 연산자 | 의미 |
|--------|------|
| & | AND |
| \| | OR |
| ! | NOT |

---

## 예제 1. AND

```r
age <- 25
member <- TRUE

if(age >= 20 & member){

    print("입장 가능")

}
```

결과

```
[1] "입장 가능"
```

두 조건이 모두 TRUE여야 한다.

---

## 예제 2. OR

```r
vip <- FALSE
coupon <- TRUE

if(vip | coupon){

    print("할인 적용")

}
```

결과

```
[1] "할인 적용"
```

둘 중 하나만 TRUE여도 실행된다.

---

## 예제 3. NOT

```r
login <- FALSE

if(!login){

    print("로그인이 필요합니다.")

}
```

결과

```
[1] "로그인이 필요합니다."
```

---

# 중첩 조건문

조건문 안에 또 다른 조건문을 작성할 수도 있다.

```r
score <- 95
attendance <- TRUE

if(score >= 90){

    if(attendance){

        print("장학생")

    }

}
```

결과

```
[1] "장학생"
```

중첩 조건문은 복잡해질 수 있으므로 필요한 경우에만 사용하는 것이 좋다.

---

# ifelse() 함수

벡터 전체를 대상으로 조건을 적용할 때는 `ifelse()`를 많이 사용한다.

## 문법

```r
ifelse(조건, 참일 때, 거짓일 때)
```

---

## 예제

```r
score <- c(95, 82, 65, 100)

ifelse(score >= 80,
       "합격",
       "불합격")
```

결과

```
[1] "합격"   "합격"   "불합격" "합격"
```

`ifelse()`는 데이터 분석에서 매우 자주 사용된다.

---

# 실무 예제 1. 할인 금액 계산

```r
price <- 150000

if(price >= 100000){

    discount <- price * 0.1

}else{

    discount <- 0

}

discount
```

결과

```
[1] 15000
```

# 실무 예제 2. 결측치 확인

```r
value <- NA

if(is.na(value)){

    print("결측치입니다.")

}else{

    print("정상 데이터입니다.")

}
```

결과

```
[1] "결측치입니다."
```

---

# 실무 예제 3. 데이터 등급 만들기

```r
sales <- c(100, 350, 700, 1200)

grade <- ifelse(
    sales >= 500,
    "High",
    "Low"
)

grade
```

결과

```
[1] "Low"  "Low"  "High" "High"
```

# 조건문 사용 시 주의사항

### 1. 비교는 `=`이 아니라 `==`를 사용한다.

```r
score == 90
```

`=`는 값을 저장하는 연산자이다.

---

### 2. 조건은 TRUE 또는 FALSE를 반환해야 한다.

```r
score > 80
```

---

### 3. 벡터에는 if보다 ifelse()를 사용하는 것이 좋다.

```r
ifelse(score >= 80,
       "Pass",
       "Fail")
```

---

# 자주 사용하는 조건 함수

| 함수 | 설명 |
|------|------|
| is.na() | 결측치 확인 |
| is.null() | NULL 확인 |
| is.numeric() | 숫자형 확인 |
| is.character() | 문자형 확인 |
| is.factor() | Factor 확인 |

# 마무리

조건문은 프로그램의 실행 흐름을 제어하는 가장 기본적인 문법이며, 데이터 분석에서도 매우 자주 사용된다.

특히 다음 세 가지는 반드시 익혀두는 것이 좋다.

- `if`
- `if ~ else`
- `ifelse()`
