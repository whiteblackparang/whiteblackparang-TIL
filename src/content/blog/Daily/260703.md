# 웹의 동작 개념

- 브라우저는 서버에 **요청(Request)** 을 보내고, 서버로부터 받은 **HTML, CSS, JavaScript**를 해석하여 화면을 그린다.
- 서버는 미리 만들어 둔 **API**를 통해 요청을 받아 처리한 뒤, HTML뿐만 아니라 **JSON**과 같은 데이터 형식으로도 응답할 수 있다.
- JavaScript는 API에서 받은 JSON 데이터를 이용하여 페이지 전체를 새로고침하지 않고도 화면 일부를 실시간으로 변경할 수 있다.

# HTML

HTML(HyperText Markup Language)은 **태그(Tag)** 를 사용하여 웹페이지의 구조를 만드는 마크업 언어이다.

사진, 텍스트, 링크 등 다양한 요소를 배치할 수 있다.

태그는 대부분 시작 태그와 종료 태그로 이루어진다.

```html
<p>안녕하세요.</p>
```

## HTML 문서 구조

HTML 문서는 크게 `head`와 `body`로 구성된다.

### head

웹페이지의 정보를 작성하는 영역이다.

주로 사용하는 태그

- `meta`
- `title`
- `link`
- `script`

### body

사용자에게 실제로 보이는 화면을 작성하는 영역이다.

주로 사용하는 태그

- `div`
- `p`
- `img`
- `a`
- `button`

---

## HTML의 트리(Tree) 구조

HTML은 부모와 자식 관계를 가지는 **트리(Tree) 구조**로 이루어진다.

```html
<div>
    <p>안녕하세요.</p>
</div>
```

위 구조에서

- `div` : 부모(Parent)
- `p` : 자식(Child)

어떤 요소 안에 어떤 요소가 포함되어 있는지가 매우 중요하다.

---

# CSS

CSS(Cascading Style Sheets)는 HTML 요소의 디자인과 스타일을 지정하는 언어이다.

스타일을 적용하기 위해서는 먼저 **선택자(Selector)** 로 원하는 요소를 선택해야 한다.

보통 여러 요소를 구분하기 위해 `class` 값을 지정한다.

```html
<div class="card"></div>
```

클래스 이름은 디자인보다는 **역할이나 의미**를 나타내도록 작성하는 것이 좋다.

CSS는 중괄호 `{}`를 사용하여 스타일을 작성한다.

```css
.card {
    color: blue;
}
```

---

## CSS 선택자

### 전체 선택자

```css
* {}
```

모든 요소를 선택한다.

### 태그 선택자

```css
p {}
```

특정 태그를 선택한다.

### 클래스 선택자

```css
.menu {}
```

클래스를 선택한다.

### 아이디 선택자

```css
#header {}
```

아이디를 선택한다.

### 태그 + 클래스 선택자

```css
div.card {}
```

`card` 클래스를 가진 `div`만 선택한다.

### 자식 선택자

```css
#header > p {}
```

`#header`의 **직계 자식**인 `p`만 선택한다.

### 자손 선택자

```css
div.card p {}
```

`card` 클래스를 가진 `div` 내부에 있는 모든 `p`를 선택한다.

---

## CSS 적용 순서

- CSS는 기본적으로 위에서 아래 순서대로 적용된다.
- 부모 요소에 적용된 일부 속성(예: `font-family`, `color`)은 자식 요소에게도 상속된다.
- 모든 CSS 속성이 상속되는 것은 아니다.

---

## 폰트 적용

웹 폰트를 사용할 때는

1. `head`에 폰트 링크(또는 임베드 코드)를 추가한다.
2. CSS에서 `font-family`를 지정한다.

```css
* {
    font-family: "Pretendard";
}
```

## 주석

### HTML

```html
<!-- 주석 -->
```

### CSS

```css
/* 주석 */
```

대부분의 코드 편집기에서는

- Windows : `Ctrl + /`
- macOS : `Cmd + /`

단축키로 주석을 쉽게 작성할 수 있다.

# Bootstrap

Bootstrap은 트위터(현 X)에서 개발한 **프론트엔드 CSS 프레임워크**이다.

미리 만들어진 CSS를 제공하기 때문에 HTML에 정해진 클래스를 작성하기만 해도 별도의 CSS 없이 다양한 디자인을 적용할 수 있다.

```html
<button class="btn btn-primary">
    버튼
</button>
```

Bootstrap은

- 버튼(Button)
- 카드(Card)
- 폼(Form)
- 네비게이션(Navigation)
- 레이아웃(Layout)

등 다양한 UI 컴포넌트를 제공하며, 반응형 웹도 기본적으로 지원한다.

---

## form과 div

`form` 태그는 사용자 입력을 서버로 전송하기 위한 태그이다.

```html
<form>
    ...
</form>
```

단순히 화면을 구분하거나 레이아웃을 구성할 때는 `div`를 사용하는 것이 좋다.

`form`은 입력값 제출 기능을 가지고 있으므로, 필요하지 않은 곳에 사용하면 의도하지 않은 동작이나 기능 충돌이 발생할 수 있다.
