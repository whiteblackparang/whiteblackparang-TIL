# JavaScript 기초 정리

## JavaScript란?

- JavaScript는 프로그래밍 언어
- 웹 브라우저에서 기본적으로 실행되는 언어
- 웹 페이지를 동적으로 만들고 사용자와 상호작용하는 기능을 구현할 수 있음

---

# 변수(Variable)

변수는 데이터를 저장하는 공간

## 변수 선언

```javascript
let name = "Tom";
let age = 20;
```

- `let`을 사용하여 변수를 선언한다.
- 선언 후 값을 변경할 때는 `let`을 다시 쓰지 않는다.

```javascript
let age = 20;

age = 21;
```

### 상수(const)

변하지 않는 값은 `const`를 사용

```javascript
const PI = 3.141592;
```

`const`로 선언한 변수는 값을 변경할 수 없다.

```javascript
const PI = 3.14;

PI = 5; // 오류 발생
```

# 자료형(Data Type)

## 숫자(Number)

```javascript
10
3.14
```

## 문자열(String)

```javascript
"Hello"
'JavaScript'
```

문자열끼리는 이어 붙여진다.

```javascript
"Hello" + " World"
```

결과

```
Hello World
```

문자열과 숫자를 더하면 숫자가 문자열로 변환된다.

```javascript
"나이 : " + 20
```

결과

```
나이 : 20
```

## Boolean

참(True)과 거짓(False)을 나타내는 자료형이다.

```javascript
true
false
```

반드시 소문자로 작성한다.

```javascript
true
false
```

잘못된 예

```javascript
True
False
```

`True`, `False`는 Boolean이 아니라 변수명으로 인식된다.

## null

값이 없음을 의도적으로 나타낸다.

```javascript
let data = null;
```

## undefined

값이 아직 할당되지 않은 상태이다.

```javascript
let data;

console.log(data);
```

결과

```
undefined
```

# 문자열 메서드

## toUpperCase()

소문자를 대문자로 변경한다.

```javascript
"hello".toUpperCase();
```

결과

```
HELLO
```

---

## toLowerCase()

대문자를 소문자로 변경한다.

```javascript
"HELLO".toLowerCase();
```

결과

```
hello
```

---

## split()

문자열을 특정 기준으로 나눈다.

```javascript
let text = "apple,banana,orange";

text.split(",");
```

결과

```javascript
["apple", "banana", "orange"]
```

---

## join()

배열의 요소를 하나의 문자열로 합친다.

```javascript
let fruits = ["apple", "banana", "orange"];

fruits.join("-");
```

결과

```
apple-banana-orange
```

---

## trim()

문자열의 앞뒤 공백을 제거한다.

```javascript
" hello ".trim();
```

결과

```
hello
```

# 증감 연산자

## 증가

```javascript
a++;
```

현재 값에 1을 더한다.

```javascript
++a;
```

현재 값에 1을 더한다.

---

## 감소

```javascript
a--;
```

현재 값에서 1을 뺀다.

```javascript
--a;
```

현재 값에서 1을 뺀다.

> `a++`와 `++a`는 모두 1을 증가시키지만, 값을 사용하는 시점이 다르다.

---

# 산술 연산자

| 연산자 | 의미 |
|---------|------|
| `+` | 더하기 |
| `-` | 빼기 |
| `*` | 곱하기 |
| `/` | 나누기 |
| `%` | 나머지 |
| `**` | 거듭제곱 |

예시

```javascript
10 % 3
```

결과

```
1
```

---

# 비교 연산자

| 연산자 | 의미 |
|---------|------|
| `<` | 작다 |
| `>` | 크다 |
| `<=` | 작거나 같다 |
| `>=` | 크거나 같다 |
| `==` | 값만 비교 |
| `===` | 값과 자료형까지 비교 |
| `!=` | 같지 않다 |
| `!==` | 값 또는 자료형이 다르다 |

예시

```javascript
10 == "10"
```

결과

```
true
```

```javascript
10 === "10"
```

결과

```
false
```

> JavaScript에서는 `==`보다 `===` 사용을 권장한다.

# 배열(Array)

배열은 여러 개의 데이터를 저장하는 자료형이다.

```javascript
let fruits = ["사과", "바나나", "포도"];
```

특징

- 순서가 있다.
- 인덱스는 0부터 시작한다.
- 서로 다른 자료형을 함께 저장할 수 있다.

```javascript
let arr = [1, "Hello", true];
```

---

## 배열 요소 접근

```javascript
fruits[0];
```

결과

```
사과
```

---

## push()

맨 뒤에 데이터를 추가한다.

```javascript
fruits.push("딸기");
```

---

## pop()

마지막 데이터를 제거한다.

```javascript
fruits.pop();
```

---

## length

배열의 길이를 반환한다.

```javascript
fruits.length;
```

---

## concat()

배열을 이어 붙인다.

```javascript
let a = [1, 2];
let b = [3, 4];

a.concat(b);
```

결과

```javascript
[1, 2, 3, 4]
```

# 객체(Object)

객체는 **key**와 **value**를 한 쌍으로 저장하는 자료형이다.

```javascript
let person = {
    name: "Tom",
    age: 20
};
```

값 가져오기

```javascript
person.name
```

또는

```javascript
person["name"]
```

객체 안에 배열을 저장할 수도 있다.

```javascript
let student = {
    name: "Kim",
    scores: [90, 80, 100]
};
```

배열 안에 객체도 저장할 수 있다.

```javascript
let students = [
    { name: "Kim", age: 20 },
    { name: "Lee", age: 22 }
];
```

# 함수(Function)

함수는 여러 개의 명령을 하나로 묶어 필요할 때 실행하는 기능이다.

## 함수 선언

```javascript
function add(a, b) {
    return a + b;
}
```

## 함수 호출

```javascript
add(3, 5);
```

결과

```
8
```

# 조건문(if)

조건에 따라 다른 코드를 실행한다.

```javascript
let age = 20;

if (age >= 20) {
    console.log("성인");
}
```

if, else 사용

```javascript
if (score >= 60) {
    console.log("합격");
} else {
    console.log("불합격");
}
```

---

# 반복문(for)

같은 작업을 여러 번 반복한다.

```javascript
for (let i = 0; i < 5; i++) {
    console.log(i);
}
```

출력

```
0
1
2
3
4
```

---

# 반복문(while)

조건이 참인 동안 반복한다.

```javascript
let i = 0;

while (i < 5) {
    console.log(i);
    i++;
}
```

# JavaScript 핵심 정리

- `let` : 변수 선언
- `const` : 상수 선언
- 문자열은 `+`로 이어 붙일 수 있다.
- Boolean은 `true`, `false`를 사용한다.
- 배열(Array)은 여러 데이터를 순서대로 저장한다.
- 객체(Object)는 `key`와 `value`를 저장한다.
- 함수(Function)는 코드를 재사용하기 위해 사용한다.
- 조건문은 `if`
- 반복문은 `for`, `while`
- 비교는 `===` 사용을 권장한다.
- 배열의 인덱스는 **0부터 시작**한다.
- JavaScript에서는 **camelCase**를 사용하는 것이 일반적이다.
