# 조건문

조건에 따라 다른 코드를 실행해야 할 때 사용한다.

## if문

조건식에는 **Boolean(true 또는 false)** 값이 들어간다.

```javascript
let age = 20;

if (age >= 20) {
    console.log("성인");
}
```

비교 연산자를 사용하여 조건을 작성할 수 있다.

```javascript
a < b
a > b
a <= b
a >= b
a === b
a !== b
```

논리 연산자도 함께 사용할 수 있다.

- `&&` : 두 조건이 모두 참일 때
- `||` : 두 조건 중 하나만 참이어도 참
- `!` : 참과 거짓을 반대로 변경

예시

```javascript
if (age >= 20 && age < 65) {
    console.log("성인");
}
```

# 반복문

같은 작업을 여러 번 반복해야 할 때 사용한다.

## for문

기본 구조

```javascript
for (초기값; 조건식; 증감식) {
    // 반복 실행할 코드
}
```

예시

```javascript
for (let i = 0; i < 5; i++) {
    console.log(i);
}
```

출력

```
0
1
2
3
4
```

반복문은 숫자뿐만 아니라 문자열이나 배열의 요소를 하나씩 처리할 때도 사용할 수 있다.

```javascript
let fruits = ["사과", "바나나", "포도"];

for (let i = 0; i < fruits.length; i++) {
    console.log(fruits[i]);
}
```

출력

```
사과
바나나
포도
```

# 문제를 풀면서 배운 점

문제를 바로 코딩하기보다 **문제를 먼저 이해하는 과정**이 중요했다.

- 어떤 순서로 연산을 수행해야 하는지 먼저 생각한다.
- 입력값과 출력값을 확인한다.
- 필요한 변수와 반복 횟수를 정리한다.

코드를 작성하기 전에 **슈도코드(Pseudocode)** 를 먼저 작성하는 습관을 들이는 것이 문제 해결에 도움이 된다.

예시

```text
1. 숫자를 입력받는다.
2. 1부터 입력한 숫자까지 반복한다.
3. 합계를 계속 더한다.
4. 결과를 출력한다.
```

이처럼 먼저 흐름을 정리한 후 코드를 작성하면 구현이 훨씬 수월하다.

---

# 막혔던 점

- 조건문과 반복문을 언제 사용해야 하는지 헷갈렸다.
- 반복문의 구조(`초기값`, `조건식`, `증감식`)를 이해하는 데 시간이 걸렸다.
- 문제를 보면 바로 코드를 작성하려는 습관이 있었다.

---

# 해결한 점

- 조건에 따라 실행이 달라질 때는 **조건문**, 같은 작업을 여러 번 수행할 때는 **반복문**을 사용한다는 점을 이해했다.
- 반복문의 실행 순서를 직접 따라가며 동작 방식을 익혔다.
- 코드를 작성하기 전에 **슈도코드로 문제 해결 과정을 먼저 정리**하는 습관을 들이기로 했다.
