# 오늘 배운 점

## jQuery

> HTML 요소를 더 쉽고 편하게 조작할 수 있도록 만든 자바스크립트 라이브러리이다.

JavaScript만으로도 구현할 수 있지만, 브라우저 호환성 문제나 복잡한 DOM 조작을 더 쉽게 해결하기 위해 많이 사용한다.

### jQuery 사용하기

- 사용하기 전에 CDN 또는 파일을 import 해주어야 한다.
- 부트스트랩처럼 보통 `</body>` 태그 바로 위에 `<script>` 태그로 추가한다.
- `$()`를 사용하여 원하는 요소를 선택한다.
- CSS는 주로 class 선택자를 많이 사용하지만, jQuery에서는 특정 요소 하나를 제어하기 위해 id 선택자를 많이 사용한다. 물론 class 선택자도 사용할 수 있다.

---

### val()

폼 요소(input, textarea 등)의 값을 가져오거나 변경할 때 사용하는 메소드이다.

---

### hide() / show()

요소를 숨기거나 다시 보여주는 메소드이다.

`hide()`를 사용하면 `display: none;` 상태가 되며, `show()`를 사용하면 다시 화면에 표시된다.

현재 display 속성을 확인하고 싶을 때는 `.css('display')`를 사용할 수 있다.

---

### text()

선택한 요소의 텍스트를 변경하는 메소드이다.

HTML 태그가 아닌 텍스트만 변경된다.

---

### append()

선택한 요소의 마지막 자식으로 새로운 요소를 추가하는 메소드이다.

기존 내용은 유지되고 뒤에 이어서 추가된다.

---

### empty()

선택한 요소는 그대로 두고 내부의 모든 자식 요소만 삭제하는 메소드이다.

---

### $(document).ready()

페이지의 모든 요소가 로딩된 후 바로 실행되어야 하는 코드를 작성할 때 사용하는 메소드이다.

---

### includes()

배열이나 문자열에 특정 값이 포함되어 있는지 확인하는 메소드이다.

포함 여부만 `true` 또는 `false`로 반환한다.

---

### split()

문자열을 원하는 기준으로 나누어 배열로 만들어주는 메소드이다.

---

## 템플릿 리터럴 (Template Literal)

템플릿 리터럴은 문자열을 백틱(`)으로 감싸 사용하는 문법이다.

### 문자열 연결

기존처럼 `+` 연산자를 사용하지 않고 문자열 안에서 변수를 바로 사용할 수 있어 코드가 훨씬 읽기 쉬워진다.

---

### 변수 출력

`${}` 문법을 사용하면 문자열 안에서 변수나 계산 결과를 바로 출력할 수 있다.

---

### 여러 줄 문자열

기존 문자열에서는 `\n`을 사용해야 줄바꿈이 가능했지만, 템플릿 리터럴은 실제 작성한 형태 그대로 여러 줄 문자열을 표현할 수 있다.

---

# 예시

### 요소 선택

- `$('#id')`
- `$('.class')`

### 값 가져오기 / 변경

- `val()`

### 요소 숨기기 / 보이기

- `hide()`
- `show()`
- `.css('display')`

### 텍스트 변경

- `text()`

### 요소 추가

- `append()`

### 내부 요소 삭제

- `empty()`

### 페이지 로딩 후 실행

- `$(document).ready(function(){ ... });`

### 포함 여부 확인

- `includes()`

### 문자열 나누기

- `split(',')`

### 템플릿 리터럴

- `` `문자열` ``
- `` `${변수}` ``
- 여러 줄 문자열 작성
