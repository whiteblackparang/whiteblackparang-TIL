# Python String (문자열)

오늘은 Python의 **문자열(String)** 에 대해 학습했다. 숫자형과 달리 문자열은 텍스트 데이터를 다루기 위한 자료형으로, Python에서 가장 자주 사용하는 데이터 타입 중 하나이다. 단순히 글자를 저장하는 것뿐만 아니라 다양한 메서드를 통해 데이터를 가공하고 분석할 수 있다는 점이 중요하다.

## 문자열(String)이란?

문자열(String)은 **문자의 연속(Sequence of Characters)** 을 의미한다. Python에서는 따옴표(`' '` 또는 `" "`)로 감싸진 데이터를 문자열로 인식한다.

문자열은 숫자처럼 계산되는 값이 아니라 텍스트 데이터를 표현하기 위한 자료형이며, 변수에 저장하여 사용할 수 있다.

## 문자열 생성

Python에서는 작은따옴표와 큰따옴표 모두 사용할 수 있다.

상황에 따라 문자열 안에 작은따옴표나 큰따옴표가 포함되어 있다면 다른 따옴표를 사용하거나 Escape 문자(`\`)를 이용하여 표현할 수 있다.

또한 여러 줄의 문자열을 작성할 때는 Triple Quotes(`'''` 또는 `"""`)를 사용할 수 있으며, 긴 설명이나 문서(Docstring)를 작성할 때 자주 활용된다.

## 문자열의 특징

문자열은 **Sequence(순서가 있는 자료형)** 이기 때문에 위치(Index)를 가진다.

Python은 **0부터 인덱스를 시작**하며, 원하는 위치의 문자 하나를 가져오거나 일정 구간을 잘라낼 수도 있다.

이러한 특성을 이용하면 문자열 검색, 일부 추출, 뒤집기 등의 다양한 작업을 수행할 수 있다.

## 문자열 연산

문자열도 연산이 가능하다.

- `+` : 문자열 연결(Concatenation)
- `*` : 문자열 반복
- `-` : 지원하지 않음

문자열끼리 이어 붙이거나 여러 번 반복하는 작업은 매우 자주 사용된다.

## 문자열 메서드

Python 문자열은 다양한 내장 메서드를 제공한다.

대표적으로

- 대소문자 변환
- 문자열 분리(split)
- 문자열 치환(replace)
- 공백 제거(strip)
- 문자열 검색(find)
- 문자열 길이(len)

등을 지원한다.

이러한 메서드를 활용하면 별도의 반복문 없이도 대부분의 문자열 처리를 간단하게 수행할 수 있다.

## 메서드 체이닝(Method Chaining)

Python의 문자열 메서드는 대부분 **새로운 문자열을 반환**한다.

따라서 하나의 메서드 실행 결과를 이어서 다음 메서드에 사용할 수 있는데 이를 **Method Chaining**이라고 한다.

복잡한 문자열 처리도 한 줄로 표현할 수 있어 Python에서 자주 사용하는 문법이다.

## 문자열 슬라이싱(Slicing)

문자열은 슬라이싱을 이용하여 원하는 부분만 추출할 수 있다.

특히 Step 값을 음수로 지정하면 문자열을 거꾸로 읽을 수 있기 때문에 문자열 뒤집기에도 활용된다.

슬라이싱은 이후 리스트(List), 튜플(Tuple) 등 다른 Sequence 자료형에서도 동일하게 사용되는 핵심 문법이다.

## 문자열 포매팅(F-String)

문자열 안에 변수나 계산 결과를 삽입해야 하는 경우 **f-string**을 사용한다.

기존의 문자열 연결 방식보다 가독성이 좋으며, Python 3.6 이후 가장 권장되는 문자열 포매팅 방식이다.

중괄호(`{}`) 안에는 변수뿐 아니라 계산식이나 함수 호출도 사용할 수 있어 매우 편리하다.

## 오늘 배운 핵심

- 문자열(String)은 문자들의 순서 있는 집합(Sequence)이다.
- 작은따옴표와 큰따옴표 모두 사용할 수 있으며, 여러 줄 문자열은 Triple Quotes를 사용한다.
- 문자열은 인덱싱과 슬라이싱이 가능하다.
- `+`, `*` 연산을 지원하지만 `-` 연산은 사용할 수 없다.
- 다양한 문자열 메서드를 통해 검색, 치환, 분리, 변환 등을 쉽게 수행할 수 있다.
- 메서드 체이닝을 활용하면 여러 문자열 처리를 한 번에 연결할 수 있다.
- f-string은 가장 직관적이고 권장되는 문자열 포매팅 방식이다.
