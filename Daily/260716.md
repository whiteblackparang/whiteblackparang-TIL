# Python print() 함수

오늘은 Python에서 가장 기본적으로 사용하는 **`print()` 함수**에 대해 학습했다. 지금까지는 REPL 환경에서 값을 입력하면 자동으로 결과가 출력되었지만, 일반적인 Python 프로그램에서는 값을 화면에 출력하기 위해 `print()` 함수를 사용해야 한다.

## print() 함수란?

`print()`는 데이터를 **표준 출력(Standard Output)** 으로 보내는 Python의 내장 함수이다.

프로그램이 실행되는 동안 사용자에게 결과를 보여주거나, 변수의 값이나 계산 결과를 확인할 때 가장 많이 사용하는 함수이다. 디버깅을 위해 매우 자주 활용된다.

## 함수(Function)란?

함수(Function)는 **특정 기능을 수행하는 코드의 묶음**이다.

필요할 때 함수 이름을 호출하면 미리 정의된 작업을 수행하며, 필요한 경우 괄호 안에 값을 전달(Argument)하여 원하는 결과를 얻을 수 있다.

`print()` 역시 Python에서 기본 제공하는 **내장 함수(Built-in Function)** 중 하나이다.

## print()의 특징

`print()`는 괄호 안에 전달한 값을 화면에 출력한다.

출력 가능한 대상은 문자열뿐 아니라 숫자, 논리값(Boolean) 등 대부분의 Python 객체이며, 여러 개의 값을 동시에 출력할 수도 있다.

또한 아무 값도 전달하지 않으면 빈 줄을 출력한다.

## 특수 문자(Escape Sequence)

`print()`는 문자열 안의 **Escape Sequence**를 실제 문자로 변환하여 출력한다.

대표적으로 다음과 같은 문자를 자주 사용한다.

- `\n` : 줄바꿈(New Line)
- `\t` : 탭(Tab)

이러한 특수 문자를 활용하면 출력 형식을 보다 보기 좋게 구성할 수 있다.

## REPL과 print()의 차이

Python REPL에서는 값을 입력하면 자동으로 결과를 보여준다.

하지만 일반 Python 파일(`.py`)에서는 표현식만 작성해도 아무것도 출력되지 않는다.

프로그램의 실행 결과를 사용자에게 보여주기 위해서는 반드시 `print()`를 사용해야 한다.

- **REPL** → 자동 출력
- **Python Script** → `print()`로 직접 출력


## 오늘 배운 핵심

- `print()`는 Python의 대표적인 내장 함수이다.
- 프로그램의 실행 결과를 화면에 출력할 때 사용한다.
- 문자열, 숫자 등 대부분의 객체를 출력할 수 있다.
- 여러 개의 값을 동시에 출력할 수 있다.
- `\n`, `\t`와 같은 Escape Sequence를 실제 문자로 변환하여 출력한다.
- REPL에서는 자동으로 출력되지만 일반 Python 프로그램에서는 `print()`를 사용해야 한다.
- 개발 과정에서는 결과 확인과 디버깅을 위해 가장 자주 사용하는 함수 중 하나이다.
