# Python Boolean과 조건문(if / else)

오늘은 Python의 **Boolean 자료형**과 **조건문(if / else)** 에 대해 학습했다. Boolean은 프로그램이 특정 조건에 따라 서로 다른 동작을 수행하도록 만드는 핵심 개념이다. 실제 프로그램은 대부분 조건에 따라 흐름이 바뀌기 때문에 Boolean과 조건문은 반드시 이해해야 하는 기본 문법이다.

## Boolean이란?

Boolean은 Python에서 가장 단순한 자료형으로 **`True` 또는 `False` 두 가지 값만** 가진다.

컴퓨터는 내부적으로 수많은 판단을 참(True) 또는 거짓(False)으로 처리하며, Python에서도 이러한 Boolean 값을 이용해 프로그램의 실행 흐름을 제어한다.

Python에서는 다른 언어와 달리 `True`, `False`의 첫 글자를 반드시 대문자로 작성해야 한다.

## 조건문(if / else)

조건문은 **주어진 조건이 참인지 거짓인지에 따라 실행할 코드를 선택하는 문법**이다.

`if` 뒤에는 True 또는 False를 반환하는 조건식이 들어가며,

- 조건이 `True`이면 해당 코드 블록이 실행되고,
- `False`이면 건너뛴다.

필요하다면 `else`를 함께 사용하여 조건이 거짓일 때 실행할 코드도 작성할 수 있다.

이를 통해 하나의 프로그램이 상황에 따라 다른 동작을 수행할 수 있다.

## 비교 연산자

조건식에서는 주로 **비교 연산자(Comparison Operator)** 를 사용한다.

대표적인 비교 연산자는 다음과 같다.

- `>` : 크다
- `<` : 작다
- `>=` : 크거나 같다
- `<=` : 작거나 같다
- `==` : 같다
- `!=` : 같지 않다

비교 연산의 결과는 항상 **Boolean(True 또는 False)** 으로 반환된다.

## 논리 연산자

여러 조건을 함께 사용할 때는 **논리 연산자(Logical Operator)** 를 사용한다.

- `and` : 모든 조건이 참일 때 True
- `or` : 하나라도 참이면 True
- `not` : 결과를 반대로 변경

논리 연산자는 복잡한 조건을 조합할 때 매우 자주 사용된다.

## 자료형 비교

같은 자료형끼리는 비교가 가능하지만, 서로 다른 자료형을 비교하면 대부분 오류가 발생한다.

예를 들어 숫자와 문자열을 비교하면 `TypeError`가 발생한다.

다만 Python에서는 Boolean이 내부적으로 정수(Integer)의 특수한 형태이기 때문에

- `True`는 `1`
- `False`는 `0`

처럼 동작하기도 한다.

이러한 특징 때문에 Boolean과 숫자를 함께 계산하는 것이 가능하지만, 코드의 가독성을 위해서는 필요한 경우가 아니라면 혼용하지 않는 것이 좋다.

## 오늘 배운 핵심

- Boolean은 `True`와 `False` 두 가지 값만 가지는 자료형이다.
- 조건문(`if`, `else`)은 Boolean 값을 이용해 프로그램의 실행 흐름을 제어한다.
- 비교 연산자의 결과는 항상 Boolean이다.
- `and`, `or`, `not`을 사용하여 여러 조건을 조합할 수 있다.
- 서로 다른 자료형 비교는 대부분 오류가 발생한다.
- Python에서는 `True`는 `1`, `False`는 `0`처럼 동작하지만, 실제 코드에서는 Boolean과 숫자를 혼용하지 않는 것이 권장된다.
