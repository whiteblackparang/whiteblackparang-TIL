# 집합 (Set)

## 집합이란?

집합(Set)은 **중복을 허용하지 않고 순서가 없는 자료형**.

- 중괄호 `{}` 사용
- 중복 요소 자동 제거
- 요소의 순서 보장 없음
- 인덱스로 요소 접근 불가
- 집합 연산 지원
- 빈 집합 생성은 `set()` 사용 (`{}`는 빈 딕셔너리)

---

## 집합 생성하기

집합을 만드는 다양한 방법.

- `{}`로 직접 생성
- `set()` 함수로 생성
- 리스트, 튜플, 문자열 등을 집합으로 변환 가능
- 변환 과정에서 중복 요소 자동 제거
- 문자열 변환 시 문자 하나씩 요소로 저장
- 빈 집합은 `set()` 사용

---

## 중복 제거 특성

집합의 가장 큰 특징.

- 동일한 값은 하나만 저장
- 중복 데이터 자동 제거
- 리스트의 중복 제거에 자주 활용
- 필요 시 `list()`로 다시 리스트 변환 가능
- 순서가 필요하면 `sorted()` 사용

---

## add() 메서드

집합에 새로운 요소 추가.

- 존재하지 않는 요소만 추가
- 이미 존재하는 값은 자동 무시
- 원본 집합 변경
- 한 번에 하나의 요소 추가

---

## remove()와 discard()

집합에서 요소 삭제.

### remove()

- 지정한 요소 삭제
- 없는 요소 삭제 시 `KeyError` 발생

### discard()

- 지정한 요소 삭제
- 없는 요소 삭제 시 오류 발생 없음
- 존재 여부가 불확실한 경우 사용

---

## clear() 메서드

집합의 모든 요소 삭제.

- 내용만 삭제
- 집합 객체는 유지
- 빈 집합으로 초기화

---

## 합집합 (union)

두 집합의 모든 요소 결합.

- `union()` 메서드 사용
- `|` 연산자 사용 가능
- 중복 요소 자동 제거
- 새로운 집합 반환
- 원본 집합 유지

---

## 교집합 (intersection)

두 집합에 공통으로 존재하는 요소 추출.

- `intersection()` 메서드 사용
- `&` 연산자 사용 가능
- 공통 요소만 포함
- 새로운 집합 반환
- 원본 집합 유지

---

## 차집합 (difference)

한 집합에만 존재하는 요소 추출.

- `difference()` 메서드 사용
- `-` 연산자 사용 가능
- `A - B`와 `B - A`의 결과는 서로 다름
- 새로운 집합 반환
- 원본 집합 유지

---

## 대칭차집합 (symmetric_difference)

한쪽 집합에만 존재하는 모든 요소 추출.

- `symmetric_difference()` 메서드 사용
- `^` 연산자 사용 가능
- 공통 요소 제외
- `(A - B) ∪ (B - A)`와 동일한 결과
- 새로운 집합 반환
- 원본 집합 유지

---

## in / not in 연산자

집합의 요소 포함 여부 확인.

- `in` : 요소 존재 여부 확인
- `not in` : 요소 미포함 여부 확인
- 반환값은 `True` 또는 `False`
- 매우 빠른 검색 속도 제공

---

## 집합 길이

집합의 요소 개수 확인.

- `len()` 함수 사용
- 중복 제거 후의 요소 개수 반환
- 리스트, 튜플, 문자열과 동일한 사용 방법

---

## 핵심 정리

| 기능 | 사용 방법 |
|------|-----------|
| 집합 생성 | `{}` 또는 `set()` |
| 빈 집합 생성 | `set()` |
| 요소 추가 | `add()` |
| 요소 삭제 | `remove()`, `discard()` |
| 모든 요소 삭제 | `clear()` |
| 합집합 | `union()`, `\|` |
| 교집합 | `intersection()`, `&` |
| 차집합 | `difference()`, `-` |
| 대칭차집합 | `symmetric_difference()`, `^` |
| 요소 포함 확인 | `in`, `not in` |
| 요소 개수 확인 | `len()` |

> **참고**
>
> 집합은 순서를 보장하지 않는 자료형. 실행 환경에 따라 출력 순서가 달라질 수 있음.
