# 함수(Function)

함수(Function)는 여러 줄의 코드를 하나로 묶어 필요할 때마다 호출하여 사용하는 기능입니다. 반복되는 코드를 줄이고, 재사용성과 유지보수성을 높이는 데 사용됩니다.

---

# 함수 정의

함수는 `def` 키워드를 사용하여 정의합니다.

```python
def 함수이름():
    실행할 코드
```

- `def` 키워드로 함수를 정의합니다.
- 함수 이름 뒤에는 반드시 괄호 `()`를 작성합니다.
- 함수 내부 코드는 들여쓰기로 구분합니다.
- 함수를 정의했다고 실행되는 것은 아니며, 호출해야 실행됩니다.

---

# 함수 호출

정의한 함수는 함수 이름 뒤에 괄호를 붙여 호출합니다.

```python
함수이름()
```

하나의 함수는 여러 번 호출할 수 있으며 호출할 때마다 같은 작업을 수행합니다.

---

# return

`return`은 함수의 실행 결과를 호출한 곳으로 반환합니다.

```python
return 값
```

특징

- 함수의 실행을 즉시 종료합니다.
- 반환값을 호출한 곳에서 사용할 수 있습니다.
- `return`이 없으면 자동으로 `None`을 반환합니다.

---

# 매개변수(Parameter)

매개변수(Parameter)는 함수가 입력값을 받기 위한 변수입니다.

```python
def 함수(매개변수):
    ...
```

함수를 호출하면서 전달하는 실제 값을 **인자(Argument)** 라고 합니다.

```python
함수(인자)
```

매개변수를 사용하면 하나의 함수로 다양한 값을 처리할 수 있습니다.

---

# 여러 개의 매개변수

함수는 여러 개의 입력값을 받을 수 있습니다.

```python
def add(a, b):
    return a + b
```

- 쉼표(`,`)로 구분하여 작성합니다.
- 전달하는 인자의 개수와 순서가 일치해야 합니다.

---

# 여러 값 반환

함수는 여러 값을 동시에 반환할 수 있습니다.

```python
return 값1, 값2
```

여러 값을 반환하면 자동으로 **튜플(tuple)** 형태가 됩니다.

호출할 때는 다음과 같이 여러 변수로 받을 수 있습니다.

```python
a, b = 함수()
```

---

# 매개변수가 없는 함수

입력이 필요하지 않은 함수도 만들 수 있습니다.

```python
def getValue():
    return 100
```

주로

- 고정값 반환
- 설정값 반환
- 초기화 작업

등에 사용됩니다.

---

# return이 없는 함수

모든 함수가 값을 반환하는 것은 아닙니다.

```python
def printMessage():
    print("Hello")
```

이처럼 작업만 수행하는 함수는 반환값이 없으며 자동으로 `None`을 반환합니다.

---

# 함수와 조건문

함수 안에서는 `if`, `elif`, `else`를 자유롭게 사용할 수 있습니다.

입력값에 따라 다른 결과를 반환할 수 있으므로 다양한 상황을 처리할 수 있습니다.

대표적인 활용 예

- 학점 계산
- 로그인 검사
- 할인 여부 판단
- 등급 분류

---

# 함수와 반복문

함수 안에서 `for`문이나 `while`문을 사용할 수 있습니다.

반복 작업을 함수로 만들면 같은 작업을 여러 번 재사용할 수 있으며 코드도 더욱 간결해집니다.

대표적인 활용 예

- 리스트 합계 계산
- 평균 계산
- 최대값 찾기
- 문자열 처리

---

# 함수 재사용

함수는 한 번 정의하면 여러 번 호출하여 사용할 수 있습니다.

같은 로직을 여러 곳에서 사용할 수 있기 때문에

- 코드 중복 감소
- 유지보수 향상
- 오류 감소

라는 장점이 있습니다.

---

# 함수 작성 순서

실무에서는 보통 다음 순서로 함수를 작성합니다.

1. 함수가 수행할 작업을 정한다.
2. 필요한 입력값(매개변수)을 정한다.
3. 처리할 코드를 작성한다.
4. `return`으로 결과를 반환한다.
5. 다양한 입력값으로 테스트한다.

---

# 함수 사용 시 주의사항

- 함수는 하나의 역할만 수행하도록 만드는 것이 좋습니다.
- 함수 이름은 동작이 드러나도록 작성합니다.
- 반복되는 코드는 함수로 분리합니다.
- 반환값이 필요하면 `return`을 사용합니다.
- 함수가 너무 길어지면 여러 개의 함수로 나누는 것이 좋습니다.

---

# 핵심 정리

- 함수는 코드를 재사용하기 위한 기능이다.
- `def`로 함수를 정의한다.
- 함수는 호출해야 실행된다.
- `return`은 값을 반환하고 함수를 종료한다.
- 매개변수는 입력을 받는다.
- 인자는 호출할 때 전달하는 실제 값이다.
- 여러 개의 매개변수를 사용할 수 있다.
- 여러 값을 반환하면 튜플로 반환된다.
- 함수 안에서 조건문과 반복문을 사용할 수 있다.
- 함수를 재사용하면 코드가 간결하고 유지보수가 쉬워진다.
