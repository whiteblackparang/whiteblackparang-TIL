# 반복문 (Loop)

반복문은 같은 작업을 여러 번 실행할 때 사용하는 문법입니다. 데이터를 하나씩 처리하거나 일정 횟수만큼 반복하는 작업에서 가장 많이 사용됩니다.

---

# 실무 흐름

1. 반복할 데이터(리스트, 문자열, 딕셔너리 등)를 준비합니다.
2. 반복문으로 데이터를 하나씩 처리합니다.
3. 필요한 값을 누적하거나 조건을 검사합니다.
4. 반복이 끝난 후 결과를 확인하고 재사용합니다.

---

# 1. for 리스트 순회

## 설명

`for`문은 리스트의 요소를 하나씩 꺼내 순서대로 실행합니다.

형식

```python
for 변수 in 리스트:
    실행할 코드
```

### 특징

- 가장 많이 사용하는 반복문
- 리스트, 튜플, 문자열 등 반복 가능한 객체에서 사용
- 반복 횟수를 직접 계산할 필요가 없음

---

# 2. for 문자열 순회

## 설명

문자열도 문자들의 집합이므로 `for`문으로 한 글자씩 가져올 수 있습니다.

### 특징

- 문자 개수만큼 반복
- 문자열은 수정할 수 없으므로 새로운 문자열을 만들어 사용

---

# 3. for 딕셔너리 순회

## 설명

딕셔너리는 기본적으로 키를 반복합니다.

필요에 따라 다음 메서드를 사용할 수 있습니다.

- `keys()`
- `values()`
- `items()`

### 특징

- 기본 반복은 Key
- `items()`를 사용하면 Key와 Value를 동시에 가져올 수 있음

---

# 4. range()

## 설명

`range()`는 일정한 숫자 범위를 만들어 반복할 때 사용합니다.

형식

```python
range(끝)

range(시작, 끝)

range(시작, 끝, 간격)
```

### 특징

- 끝 값은 포함되지 않음
- 메모리를 거의 사용하지 않음

예시

```text
range(5)

0 1 2 3 4
```

---

# 5. range와 for

## 설명

`range()`와 `for`를 함께 사용하면 원하는 횟수만큼 반복할 수 있습니다.

### 사용 목적

- n번 반복
- 인덱스 접근
- 일정한 숫자 반복

---

# 6. while

## 설명

조건이 참인 동안 계속 반복합니다.

형식

```python
while 조건:
    실행 코드
```

### 특징

- 반복 횟수를 모를 때 사용
- 반드시 조건을 변경해야 함
- 조건을 변경하지 않으면 무한 반복 발생

---

# 7. break

## 설명

`break`는 반복문을 즉시 종료합니다.

### 특징

- 가장 가까운 반복문 하나만 종료
- 조건을 만족하면 더 이상 반복하지 않음

---

# 8. continue

## 설명

`continue`는 현재 반복만 건너뛰고 다음 반복으로 이동합니다.

### 특징

- 반복은 계속 진행
- 특정 데이터만 제외할 때 사용

---

# 9. for-else

## 설명

`for`문은 `else`를 사용할 수 있습니다.

반복이 **정상적으로 끝났을 때만** `else`가 실행됩니다.

### 실행 규칙

- break 없이 종료 → else 실행
- break 발생 → else 실행 안 함

### 활용

검색 결과를 찾지 못했을 때 자주 사용됩니다.

---

# 10. while-else

## 설명

`while`문도 `else`를 사용할 수 있습니다.

조건이 False가 되어 종료되면 `else`가 실행됩니다.

### 실행 규칙

- 정상 종료 → else 실행
- break 종료 → else 실행 안 함

---

# 11. 중첩 반복문

## 설명

반복문 안에 또 다른 반복문을 넣을 수 있습니다.

### 특징

- 2차원 데이터 처리
- 모든 조합 생성
- 표 형태 데이터 처리

### 주의

중첩이 깊어질수록 실행 횟수가 크게 증가합니다.

---

# 12. 실무 반복 처리 루프

## 설명

실무에서는 단순히 반복만 하지 않습니다.

보통 다음 순서로 작업합니다.

1. 데이터 읽기
2. 조건 검사
3. 필요한 데이터만 처리
4. 합계 또는 결과 누적
5. 오류 확인
6. 결과 검증

### 자주 사용하는 패턴

- 데이터 필터링
- 합계 계산
- 평균 계산
- 상태 확인
- 로그 처리
- 파일 처리
