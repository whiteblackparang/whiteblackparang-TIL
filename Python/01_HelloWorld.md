# Hello, World 

가장 처음으로 배우는 것은 'Hello, World'를 화면에 출력한다. 파이썬에서는 print를 사용해 화면에 텍스트를 보여준다. 작은따옴표''와 큰따옴표""는 차이없다 

---

# 한글 출력하기 

- Python은 영어, 한글, 중국어 등 다양한 나라의 글자 출력 가능 
- 따옴표 안에 넣은 글자가 화면에 보임
- 이모지도 출력가능 

```python

print('안녕하세요, 파이썬')

```
---

# 여러 줄 출력하기 

print를 여러번 사용하면 여러 줄의 텍스트를 출력할 수 있다.

```python

print('첫 번째 줄')
print('두 번째 줄')
print('세 번째 줄')

```
---

# 주석 작성하기 

- 주석은 프로그램 실행에 영향을 주지 않는다
- 설명을 추가하거나 나중에 다시볼 때 누구나 이해하기 쉽게 만든다
- 한 줄 주석은 기호로
- 과도한 주석은 가독성 해치니 주의하기

#### 여러 줄 주석 작성하는 법
- 삼중 따옴표''' 또는 """로 여러 줄 문자열을 만들어 메모처럼 코드에 작성가능

# 숫자 출력하기

- 숫자 출력할 때 따옴표 필요 없음 
- 계산에 사용 가능 

```python

print(123)

```
---

# 계산결과 출력하기

- print 안에서 직접 계산 가능  
- 더하기, 빼기, 곱하기, 나누기 등 연산 가능
