# 문자열 인덱싱 & 슬라이싱

## 인덱싱 기본

- 인덱스는 0부터 시작
- 대괄호 를 사용하여 특정 위치의 문자 가져오기 가능
- 첫 번째 문자는 0, 두 번째 문자는 1

## 마지막 문자 접근 
- 마지막 문자의 인덱스는 문자열 길이 - 1
- len 함수를 사용

## 음수 인덱싱 
- 문자열의 뒤에서부터 접근 가능
- -1은 마지막 문자, -2는 마지막에서 두 번째 문자
- 길이를 계산할 필요 없이 사용가능

## 슬라이싱 기초
- 문자열의 일부분을 추출
- start 인덱스부터 end-1 인덱스까지
- end 위치는 포함되지 않는다

## 전체 슬라이싱 
- 시작과 끝을 모두 생략하면 전체 문자열을 복사
- 문자열 전체를 복사할 때 사용
