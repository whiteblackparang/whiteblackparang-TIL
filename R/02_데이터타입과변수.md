# R 프로그래밍 기초 : 데이터 타입과 변수

R은 통계 분석과 데이터 분석을 위해 만들어진 프로그래밍 언어입니다. 현재는 학계뿐 아니라 금융, 의료, 제조, IT 기업 등 다양한 산업에서 데이터 분석과 머신러닝을 수행하는 핵심 도구로 활용되고 있습니다.

R을 배우기 위해서는 가장 먼저 **데이터를 저장하는 방법**과 **데이터의 종류**, 그리고 **변수와 벡터**의 개념을 이해해야 합니다. 

---

# R의 기본 데이터 타입

프로그래밍 언어는 데이터를 다양한 형태로 저장합니다. 

R 역시 여러 가지 데이터 타입을 제공하며, 가장 기본이 되는 것은 다음과 같습니다.

- Numeric(실수형)
- Integer(정수형)
- Character(문자형)
- Logical(논리형)

예를 들어 다음과 같습니다.

```r
4.5
```

실수형(Numeric) 데이터입니다.

```r
4
```

정수처럼 보이지만 R에서는 기본적으로 Numeric으로 저장됩니다.

```r
TRUE
FALSE
```

논리형(Logical) 데이터입니다.

```r
"Hello R"
```

문자형(Character) 데이터입니다.

문자열은 큰따옴표(`" "`) 또는 작은따옴표(`' '`)로 감싸 작성합니다.

---

# 데이터 타입 확인하기

변수가 어떤 데이터 타입인지 확인할 때는 `class()` 함수를 사용합니다.

```r
x <- 28
class(x)
```

결과

```r
[1] "numeric"
```

문자형도 동일하게 확인할 수 있습니다.

```r
y <- "R is Fantastic"
class(y)
```

결과

```r
[1] "character"
```

논리형 역시 확인 가능합니다.

```r
z <- TRUE
class(z)
```

결과

```r
[1] "logical"
```

`class()`는 객체가 어떤 자료형인지 가장 많이 확인하는 함수 중 하나입니다.

---

# 데이터 구조 확인하기

자료형뿐 아니라 데이터의 구조까지 확인하고 싶다면 `str()` 함수를 사용합니다.

```r
x <- 28
str(x)
```

결과

```r
num 28
```

문자형은 다음과 같습니다.

```r
y <- "R is Fantastic"
str(y)
```

결과

```r
chr "R is Fantastic"
```

논리형은 다음과 같습니다.

```r
z <- TRUE
str(z)
```

결과

```r
logi TRUE
```

`str()` 함수는 데이터 분석을 할 때 매우 자주 사용하는 함수입니다. 데이터프레임이나 리스트처럼 복잡한 객체의 구조를 빠르게 확인할 수 있기 때문입니다.

---

# 변수(Variable)

변수는 데이터를 저장하는 공간입니다.

숫자 하나를 저장할 수도 있고, 문자열이나 벡터, 데이터프레임, 분석 결과까지 모두 변수에 저장할 수 있습니다.

R에서는 `<-` 또는 `=` 기호를 이용하여 변수를 생성합니다.

```r
x <- 42
```

또는

```r
x = 42
```

두 방법 모두 사용할 수 있지만, R에서는 일반적으로 `<-`를 사용하는 것이 관례입니다.

변수의 값을 출력하려면 변수 이름만 입력하면 됩니다.

```r
x <- 42

x
```

결과

```r
[1] 42
```

다른 변수도 생성해보겠습니다.

```r
y <- 10

y
```

결과

```r
[1] 10
```

변수끼리도 자유롭게 계산할 수 있습니다.

```r
x - y
```

결과

```r
[1] 32
```

이처럼 변수는 계산의 재사용성을 높여주며, 복잡한 분석에서도 핵심적인 역할을 합니다.

---

# 벡터(Vector)

R에서 가장 중요한 자료구조는 **벡터(Vector)**입니다.

벡터는 **동일한 데이터 타입을 여러 개 저장하는 1차원 자료구조**입니다.

벡터는 `c()` 함수(combine)를 사용하여 생성합니다.

숫자 벡터

```r
vec_num <- c(1, 10, 49)

vec_num
```

결과

```r
[1] 1 10 49
```

문자 벡터

```r
vec_chr <- c("a", "b", "c")
```

논리형 벡터

```r
vec_bool <- c(TRUE, FALSE, TRUE)
```

실수형 벡터

```r
dbl_var <- c(1, 2.5, 4.5)
```

벡터는 데이터 분석에서 가장 많이 사용하는 자료구조입니다. 여러 개의 데이터를 한 번에 저장하고 계산할 수 있기 때문에 R의 거의 모든 함수가 벡터를 기반으로 동작합니다.

---

# 벡터의 자료형 확인

벡터 역시 자료형을 확인할 수 있습니다.

```r
vec_num <- c(1, 10, 49)

typeof(vec_num)
```

결과

```r
[1] "double"
```

또는 특정 타입인지 확인할 수도 있습니다.

```r
is.numeric(vec_num)

is.logical(vec_num)

is.character(vec_num)
```

TRUE 또는 FALSE를 반환하므로 조건문에서도 자주 활용됩니다.

---

# 벡터의 형 변환

벡터 안의 모든 요소는 동일한 자료형을 가져야 합니다.

만약 서로 다른 자료형을 함께 저장하면 R은 자동으로 하나의 자료형으로 변환합니다.

예를 들어

```r
c("a", 1)
```

결과

```r
[1] "a" "1"
```

숫자가 문자형으로 자동 변환됩니다.

필요하다면 직접 형 변환도 가능합니다.

```r
x <- c(FALSE, FALSE, TRUE)

as.numeric(x)
```

결과

```r
[1] 0 0 1
```

이외에도 다음과 같은 형 변환 함수가 자주 사용됩니다.

- `as.character()`
- `as.numeric()`
- `as.integer()`
- `as.logical()`

데이터 전처리 과정에서는 자료형을 올바르게 변환하는 작업이 매우 중요하므로 반드시 익혀두는 것이 좋습니다.

# 마무리

이번 글에서는 R 프로그래밍의 가장 기본이 되는 내용을 살펴보았습니다.

핵심 내용을 정리하면 다음과 같습니다.

- R의 기본 데이터 타입은 Numeric, Character, Logical 등이 있다.
- `class()`와 `str()`로 자료형과 데이터 구조를 확인할 수 있다.
- 변수는 `<-`를 사용하여 생성하는 것이 일반적이다.
- 벡터는 `c()` 함수로 생성하며 R에서 가장 중요한 자료구조이다.
- `typeof()`와 `is.*()` 함수로 자료형을 확인할 수 있다.
- `as.numeric()`, `as.character()` 등의 함수를 이용해 자료형을 변환할 수 있다.
