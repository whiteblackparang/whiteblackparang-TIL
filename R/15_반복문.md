# 반복문(while)

## while문이란?

`while`문은 **조건이 TRUE인 동안 동일한 코드를 반복 실행하는 반복문**이다.

반복 횟수를 미리 알 수 없는 경우에 주로 사용하며, 조건이 FALSE가 되는 순간 반복이 종료된다.

예를 들어 다음과 같은 상황에서 많이 사용한다.

- 특정 조건을 만족할 때까지 반복
- 사용자 입력이 올바를 때까지 반복
- 파일을 끝까지 읽을 때까지 반복
- 데이터가 존재하는 동안 반복 처리

반대로 반복 횟수가 정해져 있다면 `for`문을 사용하는 것이 일반적이다.

---

## while문의 기본 문법

```r
while (조건) {
  실행할 코드
}
```

동작 순서는 다음과 같다.

1. 조건을 확인한다.
2. 조건이 TRUE이면 코드 블록을 실행한다.
3. 다시 조건을 확인한다.
4. 조건이 FALSE가 될 때까지 반복한다.

---

## while문의 실행 과정

예를 들어 다음 코드를 살펴보자.

```r
num <- 1

while(num <= 5){
  print(num)
  num <- num + 1
}
```

### 실행 결과

```text
[1] 1
[1] 2
[1] 3
[1] 4
[1] 5
```

### 실행 순서

초기값

```r
num = 1
```

↓

조건 확인

```r
num <= 5
```

↓

TRUE

↓

출력

```r
print(num)
```

↓

증가

```r
num <- num + 1
```

↓

다시 조건 확인

↓

FALSE가 되면 종료

# 예제 8. break 사용하기

```r
i <- 1

while(TRUE){

  print(i)

  if(i == 5){
    break
  }

  i <- i + 1

}
```

### 결과

```text
[1] 1
[1] 2
[1] 3
[1] 4
[1] 5
```

`break`는 현재 반복문을 즉시 종료한다.

# 예제 9. next 사용하기

짝수만 출력해보자.

```r
i <- 0

while(i < 10){

  i <- i + 1

  if(i %% 2 != 0){
    next
  }

  print(i)

}
```

### 결과

```text
[1] 2
[1] 4
[1] 6
[1] 8
[1] 10
```

`next`는 현재 반복을 건너뛰고 다음 반복으로 이동한다.

# while 사용 시 주의사항

가장 흔한 실수는 **무한 반복(Infinite Loop)** 이다.

다음 코드는 종료되지 않는다.

```r
i <- 1

while(i <= 5){

  print(i)

}
```

### 문제점

`i`의 값이 변하지 않기 때문에 조건이 항상 TRUE이다.

올바른 코드는 다음과 같다.

```r
i <- 1

while(i <= 5){

  print(i)

  i <- i + 1

}
```

반복 변수는 반드시 변경되어야 한다.

---

# for문과 while문의 차이

| 구분 | for | while |
|------|------|--------|
| 반복 횟수 | 미리 알고 있음 | 조건에 따라 달라짐 |
| 사용 빈도 | 매우 높음 | 조건 반복 시 사용 |
| 가독성 | 높음 | 조건이 복잡할수록 낮아짐 |
| 대표 사례 | 벡터 순회 | 입력 검증, 조건 반복 |

예를 들어 100번 반복한다면 `for`문이 더 적합하다.

반면, 사용자가 올바른 값을 입력할 때까지 반복하는 경우에는 `while`문이 적합하다.

---

# 정리

- `while`은 조건이 TRUE인 동안 반복 실행한다.
- 반복 횟수를 미리 알 수 없는 경우에 적합하다.
- 반복 변수는 반드시 변경해야 한다.
- `break`는 반복을 즉시 종료한다.
- `next`는 현재 반복을 건너뛰고 다음 반복을 수행한다.
- 반복 횟수가 정해져 있다면 `for`문을 사용하는 것이 일반적이다.
