# R 소개와 데이터 분석 언어의 특징

R은 **1993년 Ross Ihaka와 Robert Gentleman**이 개발한 오픈소스 프로그래밍 언어

통계 분석과 데이터 시각화를 목적

현재는 학계뿐만 아니라 Google, Uber, Airbnb, Facebook 등 다양한 글로벌 기업에서도 데이터 분석과 머신러닝 업무에 활용되고 있다.

Python이 범용 프로그래밍 언어라면, R은 처음부터 **데이터 분석을 위해 설계된 언어**라는 점이 가장 큰 특징이다. 회귀분석, 시계열 분석, 통계적 추론, 머신러닝 알고리즘 등 다양한 분석 기법을 지원하며, 풍부한 패키지 생태계를 통해 대부분의 데이터 분석 작업을 수행할 수 있다.

---

## R의 주요 활용 분야

R은 데이터 분석 전반에서 사용되며 대표적인 활용 분야는 다음과 같다.

- 통계 분석(Statistical Analysis)
- 데이터 분석(Data Analysis)
- 머신러닝(Machine Learning)
- 데이터 시각화(Data Visualization)
- 예측 모델링 및 통계적 추론

특히 통계 기반 분석이 필요한 의료, 금융, 정부기관, 연구기관에서 높은 활용도를 보이며, 학술 연구 분야에서는 사실상 표준 도구 중 하나로 자리 잡고 있다.

---

## 데이터 분석 Workflow

R은 단순히 데이터를 계산하는 언어가 아니라 데이터 분석의 전체 과정을 지원한다.

### 1. Program
데이터를 불러오고 필요한 분석 코드를 작성한다.

### 2. Transform
데이터를 정제(Cleaning)하고 원하는 형태로 변환한다.

### 3. Discover
데이터를 탐색하며 패턴을 찾고 가설을 검증한다.

### 4. Model
통계 모델이나 머신러닝 모델을 구축하여 데이터를 분석한다.

### 5. Communicate
분석 결과를 보고서나 대시보드 형태로 공유한다.

R은 이러한 분석 과정을 하나의 환경에서 모두 수행할 수 있다는 점이 큰 장점이다.

---

## R 패키지 생태계

R의 강점 중 하나는 방대한 패키지 생태계이다.

패키지는 특정 기능을 쉽게 사용할 수 있도록 만들어진 라이브러리이며, 대부분 **CRAN(Comprehensive R Archive Network)**을 통해 무료로 제공된다.

대표적인 분야는 다음과 같다.

- 데이터 전처리
- 데이터 시각화
- 통계 분석
- 머신러닝
- 시계열 분석
- 보고서 작성

필요한 기능을 패키지 하나만 설치하여 사용할 수 있기 때문에 R의 활용 범위는 매우 넓다.

---

## R Markdown과 Shiny

R은 분석 결과를 단순히 출력하는 것에서 끝나지 않는다.

### R Markdown

코드와 결과, 그래프, 설명을 하나의 문서로 작성할 수 있는 기능이다.

다음과 같은 형식으로 변환할 수 있다.

- HTML
- PDF
- Word

분석 보고서를 자동으로 생성할 수 있어 실무에서도 많이 사용된다.

### Shiny

Shiny는 R로 웹 애플리케이션과 대시보드를 만들 수 있는 프레임워크이다.

복잡한 웹 개발 지식 없이도 데이터를 시각화하고 사용자와 상호작용하는 대시보드를 제작할 수 있다.

---

## 왜 R을 사용하는가?

데이터 분석에는 Excel, Power BI, Tableau, SAS, Python 등 다양한 도구가 존재한다.

각 도구마다 장점이 있지만 R은 다음과 같은 강점을 가진다.

- 통계 분석 기능이 매우 강력하다.
- 데이터 시각화 기능이 뛰어나다.
- 다양한 머신러닝 알고리즘을 지원한다.
- 오픈소스라 무료로 사용할 수 있다.
- 수많은 패키지를 통해 기능을 계속 확장할 수 있다.

특히 분석 결과를 문서나 웹 애플리케이션으로 바로 공유할 수 있다는 점은 다른 분석 도구와 차별화되는 장점이다.

---

## R과 Python 비교

데이터 분석 분야에서는 R과 Python이 가장 많이 비교된다.

### Python

- 범용 프로그래밍 언어
- 인공지능과 딥러닝 분야에서 강점
- 다양한 개발 환경에서 활용 가능

### R

- 통계 분석에 최적화
- 데이터 시각화 기능이 뛰어남
- 분석 보고서 작성이 편리함
- 통계 모델링 기능이 풍부함

두 언어 모두 데이터 과학에서 중요한 도구이며, 프로젝트 목적에 따라 적절하게 선택하는 것이 중요하다.

---

## R은 어려운 언어일까?

과거에는 R의 문법이 다소 복잡하고 일관성이 부족하다는 평가가 있었다.

하지만 **Hadley Wickham**이 개발한 **tidyverse**가 등장하면서 데이터 처리 방식이 크게 개선되었다.

대표적인 패키지는 다음과 같다.

- dplyr
- tidyr
- ggplot2
- readr
- stringr

이 패키지들은 동일한 문법 체계를 사용하여 데이터 전처리와 시각화를 훨씬 직관적으로 수행할 수 있도록 만들어졌다.

또한 R은 C, C++, Java와 연동할 수 있으며 Spark, Hadoop 같은 빅데이터 플랫폼도 지원한다. 최근에는 병렬 연산 기능도 강화되어 대용량 데이터 처리 성능 역시 지속적으로 향상되고 있다.

# 핵심 정리

- R은 통계 분석과 데이터 시각화를 위해 개발된 오픈소스 언어이다.
- 데이터 분석 전 과정(수집 → 전처리 → 분석 → 모델링 → 공유)을 지원한다.
- CRAN을 통해 수많은 패키지를 무료로 사용할 수 있다.
- R Markdown과 Shiny를 이용하면 보고서와 웹 애플리케이션 제작이 가능하다.
- Python과 함께 데이터 과학 분야에서 가장 널리 사용되는 언어이다.
- tidyverse의 등장으로 데이터 처리와 시각화가 더욱 직관적이고 생산적으로 발전하였다.
