# 데이터 프레임(Data Frame) : 생성, 추가, 선택, 부분집합

## 들어가며

실제 데이터 분석에서는 하나의 값만 다루는 경우보다 여러 종류의 데이터를 하나의 표(Table) 형태로 관리하는 경우가 대부분이다.

예를 들어 쇼핑몰의 상품 데이터에는 상품번호, 상품명, 가격, 재고, 판매 여부 등 여러 종류의 정보가 함께 저장된다.

|상품번호|상품명|가격|재고|
|-------|------|----|----|
|101|Book|12000|15|
|102|Pen|1500|80|
|103|Notebook|3500|32|

이처럼 **행(Row)과 열(Column)로 구성된 데이터 구조**를 R에서는 **데이터 프레임(Data Frame)** 이라고 한다.

데이터 프레임은 R에서 가장 많이 사용하는 자료구조이며, 대부분의 데이터 분석과 머신러닝은 데이터 프레임을 중심으로 이루어진다.

이번 튜토리얼에서는 다음 내용을 학습한다.

- 데이터 프레임이란?
- data.frame()으로 생성하기
- 열 이름 변경하기
- 데이터 구조 확인하기
- 행과 열 선택하기
- 새로운 열 추가하기
- 특정 열 선택하기
- 조건에 맞는 데이터 추출하기

# 데이터 프레임(Data Frame)이란?

데이터 프레임은 **같은 길이를 가진 여러 개의 벡터(Vector)를 하나의 표 형태로 묶은 자료구조**이다.

행(Row)은 하나의 데이터(레코드)를 의미하고,

열(Column)은 하나의 변수(Variable)를 의미한다.

예를 들어 학생 정보를 저장한다면 다음과 같은 형태가 된다.

|학번|이름|나이|합격|
|----|----|----|----|
|101|Kim|20|TRUE|
|102|Lee|22|FALSE|
|103|Park|21|TRUE|

데이터 프레임의 가장 큰 특징은 **열마다 서로 다른 자료형을 가질 수 있다는 것**이다.

예를 들어

- 학번 → 숫자(Numeric)
- 이름 → 문자(Character)
- 합격 여부 → 논리(Logical)

처럼 서로 다른 자료형을 함께 저장할 수 있다.

이는 모든 데이터가 동일한 자료형이어야 하는 **행렬(Matrix)** 과 가장 큰 차이점이다.

---

# 데이터 프레임 생성하기

데이터 프레임은 **data.frame()** 함수를 이용한다.

기본 문법은 다음과 같다.

```R
data.frame(변수1, 변수2, 변수3)
```

예제를 살펴보자.

```R
id <- c(10,20,30,40)
item <- c("Book","Pen","Notebook","Pencil")
store <- c(TRUE,FALSE,TRUE,FALSE)
price <- c(2.5,8,10,7)

df <- data.frame(id,item,store,price)

df
```

결과

```
   id      item   store price
1  10      Book   TRUE   2.5
2  20       Pen  FALSE   8.0
3  30  Notebook   TRUE  10.0
4  40    Pencil  FALSE   7.0
```

각 벡터가 하나의 열(Column)이 되어 데이터 프레임이 생성된다.

> **Tip**
>
> 하나의 열에 들어가는 데이터 개수는 반드시 동일해야 한다. 길이가 다르면 데이터 프레임을 만들 수 없다.

# 열(Column) 이름 변경하기

기본적으로 열 이름은 변수명을 사용한다.

하지만 분석 목적에 맞게 이름을 변경할 수도 있다.

```R
names(df) <- c(
"ID",
"Item",
"Store",
"Price"
)

df
```

결과

```
   ID      Item  Store Price
1  10      Book   TRUE   2.5
2  20       Pen  FALSE   8.0
3  30  Notebook   TRUE  10.0
4  40    Pencil  FALSE   7.0
```

열 이름은 분석 과정에서 계속 사용되므로 이해하기 쉬운 이름을 사용하는 것이 좋다.

현재 열 이름만 확인하려면 다음과 같이 사용할 수 있다.

```R
names(df)
```

# 데이터 구조 확인하기

데이터 프레임의 구조는 **str()** 함수로 확인한다.

```R
str(df)
```

결과

```
'data.frame': 4 obs. of 4 variables:

$ ID    : num
$ Item  : Factor
$ Store : logi
$ Price : num
```

여기서 확인할 수 있는 정보는 다음과 같다.

- 데이터 개수(행)
- 변수 개수(열)
- 각 열의 자료형
- Factor 여부

데이터를 불러온 직후 가장 먼저 실행하는 함수 중 하나이다.

# 데이터 선택하기 (Indexing)

데이터 프레임은

```
데이터프레임[행, 열]
```

형태로 접근한다.

예를 들어

```R
df[1,2]
```

은

**1행 2열**을 의미한다.

결과

```
Book
```

---

## 여러 행 선택하기

1~2행만 선택

```R
df[1:2, ]
```

결과

```
ID Item Store Price

10 Book TRUE 2.5

20 Pen FALSE 8
```

행만 선택하고 열을 비워두면 **모든 열**이 출력된다.

---

## 특정 열 선택하기

첫 번째 열

```R
df[,1]
```

결과

```
10
20
30
40
```

---

## 여러 행과 여러 열 선택하기

```R
df[1:3,3:4]
```

결과

```
Store Price

TRUE 2.5

FALSE 8

TRUE 10
```

---

## 열 이름으로 선택하기

번호 대신 이름을 사용하는 것이 훨씬 많이 사용된다.

```R
df[,c("ID","Store")]
```

결과

```
ID Store

10 TRUE

20 FALSE

30 TRUE

40 FALSE
```

실무에서는 열 번호보다 열 이름을 사용하는 것이 일반적이다.

# 새로운 열(Column) 추가하기

데이터 프레임에는 새로운 열을 쉽게 추가할 수 있다.

가장 많이 사용하는 방법은 **$ 연산자**이다.

```R
quantity <- c(10,35,40,5)

df$Quantity <- quantity

df
```

결과

```
ID Item Store Price Quantity

10 Book TRUE 2.5 10

20 Pen FALSE 8 35

30 Notebook TRUE 10 40

40 Pencil FALSE 7 5
```

새로운 열이 마지막에 추가된다.

> **주의**
>
> 추가하는 벡터의 길이는 데이터 프레임의 행 개수와 반드시 같아야 한다.

길이가 다르면 다음과 같은 오류가 발생한다.

```R
quantity <- c(10,20,30)

df$Quantity <- quantity
```

```
replacement has 3 rows,
data has 4
```

---

# cbind()를 이용한 열 추가

또 다른 방법은 **cbind()** 함수이다.

```R
discount <- c(1000,500,1500,300)

df <- cbind(df, discount)

df
```

결과

```
ID Item Store Price Quantity discount
```

처럼 새로운 열이 오른쪽에 추가된다.

---

# 특정 열만 선택하기

특정 열 하나만 사용할 경우에는 **$ 연산자**를 가장 많이 사용한다.

```R
df$Price
```

결과

```
2.5
8
10
7
```

또는

```R
df$Item
```

처럼 원하는 열을 바로 가져올 수 있다.

데이터 분석에서는 특정 변수의 평균이나 합계를 계산할 때 자주 사용된다.

---

# subset()을 이용한 부분집합 만들기

조건에 맞는 데이터만 선택할 때는 **subset()** 함수를 사용한다.

기본 문법은 다음과 같다.

```R
subset(
데이터프레임,
조건
)
```

예를 들어 가격이 5보다 큰 상품만 선택하려면

```R
subset(df,
       subset = Price > 5)
```

결과

```
ID Item Store Price

20 Pen FALSE 8

30 Notebook TRUE 10

40 Pencil FALSE 7
```

---

## 여러 조건 사용하기

AND 조건

```R
subset(
df,
Price > 5 &
Store == TRUE
)
```

OR 조건

```R
subset(
df,
Price > 8 |
Store == FALSE
)
```

논리 연산자를 함께 사용하면 원하는 데이터만 쉽게 추출할 수 있다.

# 데이터 프레임에서 자주 사용하는 함수

|함수|설명|
|----|----|
|data.frame()|데이터 프레임 생성|
|names()|열 이름 확인 및 변경|
|str()|구조 확인|
|head()|앞부분 데이터 확인|
|tail()|뒷부분 데이터 확인|
|summary()|기초 통계 확인|
|subset()|조건에 맞는 데이터 선택|
|cbind()|열 추가|
|rbind()|행 추가|

특히 **head()**, **str()**, **summary()**는 데이터를 불러온 직후 가장 먼저 실행하는 함수들이므로 반드시 익혀두는 것이 좋다.

---

# Matrix와 Data Frame의 차이

|Matrix|Data Frame|
|------|----------|
|같은 자료형만 저장 가능|여러 자료형 저장 가능|
|수치 계산 중심|데이터 분석 중심|
|행렬 연산에 적합|실무 데이터 처리에 적합|
|Numeric 위주|Numeric, Character, Factor, Logical 모두 가능|

대부분의 데이터 분석에서는 Matrix보다 **Data Frame**을 훨씬 많이 사용한다.

---

# 정리

- 데이터 프레임은 여러 개의 벡터를 하나의 표 형태로 관리하는 자료구조이다.
- data.frame() 함수로 쉽게 생성할 수 있다.
- names() 함수를 이용하여 열 이름을 변경할 수 있다.
- str() 함수는 데이터 구조와 자료형을 확인하는 데 매우 유용하다.
- `[행, 열]` 문법으로 원하는 데이터를 자유롭게 선택할 수 있다.
- `$`와 `cbind()`를 이용하여 새로운 열을 추가할 수 있다.
- subset() 함수는 조건에 맞는 데이터만 추출할 때 가장 많이 사용하는 함수이다.
