# 리스트(List): 생성과 요소 선택

## 리스트(List)란?

리스트(List)는 **R에서 가장 유연한 자료구조**이다.

벡터(Vector)는 하나의 자료형만 저장할 수 있지만, 리스트는 **서로 다른 자료형과 객체를 하나의 변수에 저장**할 수 있다.

예를 들어 하나의 리스트 안에는 다음과 같은 다양한 객체를 함께 저장할 수 있다.

- 숫자 벡터(Vector)
- 문자 벡터(Character)
- 논리형(Logical)
- 행렬(Matrix)
- 데이터 프레임(Data Frame)
- 함수(Function)
- 또 다른 리스트(List)

리스트는 **여러 종류의 데이터를 하나의 객체로 관리할 수 있는 컨테이너(Container)** 역할을 한다.

> **실무에서는 모델 결과, 함수의 반환값, API(JSON) 데이터 등을 저장할 때 리스트가 매우 자주 사용된다.**

---

# 리스트 생성하기

리스트는 `list()` 함수를 이용하여 생성한다.

## 문법

```r
list(element1, element2, ...)
```

### 주요 인수

|인수|설명|
|---|---|
|element1|저장할 첫 번째 객체|
|...|추가로 저장할 객체|

---

## 예제 1. 서로 다른 자료형 저장하기

```r
my_list <- list(
  100,
  "R Programming",
  TRUE
)

my_list
```

출력

```text
[[1]]
[1] 100

[[2]]
[1] "R Programming"

[[3]]
[1] TRUE
```

숫자, 문자, 논리형이 하나의 리스트 안에 함께 저장된다.

---

# 다양한 객체 저장하기

리스트에는 단순한 값뿐 아니라 벡터, 행렬, 데이터 프레임도 저장할 수 있다.

## 예제 2. 벡터 저장

```r
vect <- 1:5

my_list <- list(vect)

my_list
```

출력

```text
[[1]]
[1] 1 2 3 4 5
```

---

## 예제 3. 행렬 저장

```r
mat <- matrix(1:10, nrow = 2)

my_list <- list(mat)

my_list
```

출력

```text
[[1]]
     [,1] [,2] [,3] [,4] [,5]
[1,]    1    3    5    7    9
[2,]    2    4    6    8   10
```

---

## 예제 4. 데이터 프레임 저장

```r
df <- head(mtcars)

my_list <- list(df)

my_list
```

출력

```text
[[1]]
                   mpg cyl disp  hp drat    wt  qsec vs am gear carb
Mazda RX4         21.0   6  160 110 3.90 2.620 16.46  0  1    4    4
Mazda RX4 Wag     21.0   6  160 110 3.90 2.875 17.02  0  1    4    4
...
```

---

# 여러 객체를 하나의 리스트에 저장하기

실제로는 여러 객체를 함께 저장하는 경우가 많다.

## 예제 5

```r
vect <- 1:5
mat <- matrix(1:10, nrow = 2)
df <- head(mtcars)

my_list <- list(vect, mat, df)

my_list
```

출력

```text
[[1]]
[1] 1 2 3 4 5

[[2]]
...

[[3]]
...
```

---

# 이름(Name) 지정하기

리스트 요소에는 이름을 지정할 수 있다.

## 예제 6

```r
my_list <- list(
  numbers = 1:5,
  matrix = matrix(1:6, nrow = 2),
  data = head(mtcars)
)

my_list
```

출력

```text
$numbers
[1] 1 2 3 4 5

$matrix
...

$data
...
```

이름을 지정하면 원하는 요소를 쉽게 찾을 수 있다.

---

# 리스트 요소 선택하기

리스트는 `[[ ]]` 또는 `$`를 이용하여 요소를 선택한다.

---

## 1. 위치(Index)로 선택

```r
my_list[[1]]
```

출력

```text
[1] 1 2 3 4 5
```

첫 번째 요소를 반환한다.

---

## 2. 이름(Name)으로 선택

```r
my_list$data
```

또는

```r
my_list[["data"]]
```

출력

```text
                   mpg cyl disp hp ...
Mazda RX4         ...
```

이름을 이용하면 가독성이 좋아진다.

---

# []와 [[ ]]의 차이

초보자가 가장 많이 헷갈리는 부분이다.

## 예제 7

```r
my_list <- list(1:3, letters[1:3])

my_list[1]
```

출력

```text
[[1]]
[1] 1 2 3
```

`[]`는 **리스트 자체를 반환**한다.

---

```r
my_list[[1]]
```

출력

```text
[1] 1 2 3
```

`[[ ]]`는 **리스트 안의 실제 데이터를 반환**한다.

---

## 비교

|표현|반환 결과|
|---|---|
|`my_list[1]`|리스트(List)|
|`my_list[[1]]`|첫 번째 요소|
|`my_list$data`|이름으로 요소 선택|

실무에서는 대부분 **`[[ ]]` 또는 `$`**를 사용한다.

---

# 리스트 구조 확인하기

`str()` 함수를 이용하면 리스트 내부 구조를 쉽게 확인할 수 있다.

## 예제 8

```r
str(my_list)
```

출력

```text
List of 3
 $ numbers: int [1:5] 1 2 3 4 5
 $ matrix : int [1:2, 1:3] 1 2 3 4 5 6
 $ data   :'data.frame': 6 obs. of 11 variables
```

`str()`은 객체의 구조를 빠르게 파악할 때 매우 유용하다.

---

# 리스트에 새로운 요소 추가하기

리스트는 새로운 요소를 자유롭게 추가할 수 있다.

## 예제 9

```r
my_list$language <- "R"

my_list
```

출력

```text
$numbers
...

$matrix
...

$data
...

$language
[1] "R"
```

---

# 실무에서 리스트는 언제 사용할까?

데이터 분석에서는 데이터 프레임을 가장 많이 사용하지만, 리스트도 매우 중요한 자료구조이다.

대표적인 활용 사례는 다음과 같다.

- 여러 객체를 하나로 관리할 때
- 함수가 여러 결과를 반환할 때
- 회귀분석(`lm()`), 일반화 선형모형(`glm()`) 등의 모델 결과 저장
- 머신러닝 모델 객체 저장
- JSON/API 데이터 처리

예를 들어 회귀분석 결과는 리스트 형태로 반환된다.

```r
model <- lm(mpg ~ wt, data = mtcars)

class(model)
```

출력

```text
[1] "lm"
```

객체의 구조를 확인하면 리스트 형태임을 알 수 있다.

```r
str(model)
```

# 정리

- 리스트는 서로 다른 자료형을 함께 저장할 수 있는 자료구조이다.
- `list()` 함수로 생성한다.
- 벡터, 행렬, 데이터 프레임, 함수, 리스트 등을 함께 저장할 수 있다.
- 요소 선택은 `[[ ]]` 또는 `$`를 사용한다.
- `[]`는 리스트를 반환하고 `[[ ]]`는 실제 요소를 반환한다.
- 모델 결과, 함수 반환값, JSON 데이터 등 실무에서도 매우 자주 사용된다.
