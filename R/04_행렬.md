# Tutorial 04. 행렬(Matrix): 생성, 출력, 열 추가, 분할

## 들어가며

벡터가 **하나의 방향(1차원)** 으로 데이터를 저장한다면, 이번에 배울 **행렬(Matrix)** 은 **행(Row)과 열(Column)을 가지는 2차원 자료구조**입니다.

행렬은 데이터 분석과 통계, 머신러닝에서 매우 자주 등장합니다. 특히 여러 개의 변수와 관측값을 동시에 다룰 때 기본이 되는 자료구조이며, 이후 배우게 될 데이터 프레임(Data Frame)의 개념을 이해하는 데에도 큰 도움이 됩니다.

---

# 행렬(Matrix)이란?

행렬(Matrix)은 **동일한 자료형을 가지는 데이터들을 행(Row)과 열(Column)의 형태로 저장하는 2차원 배열**입니다.

예를 들어 다음과 같은 형태가 하나의 행렬입니다.

|     | Col1 | Col2 |
|----:|-----:|-----:|
|Row1|1|2|
|Row2|3|4|
|Row3|5|6|

행렬은 반드시 **모든 데이터가 같은 타입**이어야 합니다.

즉,

- 숫자만 저장하거나
- 문자만 저장하거나
- 논리형(TRUE/FALSE)만 저장할 수 있습니다.

---

# matrix() 함수로 행렬 생성하기

행렬은 `matrix()` 함수를 이용하여 생성합니다.

기본 문법은 다음과 같습니다.

```r
matrix(data, nrow, ncol, byrow = FALSE)
```

각 인수의 의미는 다음과 같습니다.

|인수|설명|
|---|---|
|data|행렬에 들어갈 데이터|
|nrow|행(Row)의 개수|
|ncol|열(Column)의 개수|
|byrow|행 우선 입력 여부|

---

## byrow = FALSE (기본값)

기본적으로 R은 **열(Column) 방향으로 데이터를 채웁니다.**

```r
matrix_a <- matrix(1:10,
                   nrow = 5,
                   byrow = FALSE)

matrix_a
```

결과

```text
     [,1] [,2]
[1,]    1    6
[2,]    2    7
[3,]    3    8
[4,]    4    9
[5,]    5   10
```

1부터 10까지의 숫자가 **위에서 아래 방향**으로 먼저 채워지는 것을 확인할 수 있습니다.

---

## byrow = TRUE

행 기준으로 입력하고 싶다면 `byrow = TRUE`를 사용합니다.

```r
matrix_b <- matrix(1:10,
                   nrow = 5,
                   byrow = TRUE)

matrix_b
```

결과

```text
     [,1] [,2]
[1,]    1    2
[2,]    3    4
[3,]    5    6
[4,]    7    8
[5,]    9   10
```

이번에는 숫자가 **왼쪽에서 오른쪽**으로 채워집니다.

---

## ncol을 이용하여 생성하기

행 개수 대신 열 개수를 지정할 수도 있습니다.

```r
matrix_c <- matrix(1:12,
                   ncol = 3)

matrix_c
```

결과

```text
     [,1] [,2] [,3]
[1,]    1    5    9
[2,]    2    6   10
[3,]    3    7   11
[4,]    4    8   12
```

---

# 행렬의 크기 확인하기

## dim()

행과 열을 동시에 확인합니다.

```r
dim(matrix_c)
```

결과

```r
[1] 4 3
```

첫 번째 숫자는 행의 개수,

두 번째 숫자는 열의 개수입니다.

---

## nrow()

행(Row)의 개수를 확인합니다.

```r
nrow(matrix_c)
```

결과

```r
[1] 4
```

---

## ncol()

열(Column)의 개수를 확인합니다.

```r
ncol(matrix_c)
```

결과

```r
[1] 3
```

---

# dim()을 이용하여 행렬 만들기

벡터에 차원 정보를 부여하면 행렬이 됩니다.

```r
x <- 1:6

dim(x) <- c(3,2)

x
```

결과

```text
     [,1] [,2]
[1,]    1    4
[2,]    2    5
[3,]    3    6
```

---

# 열(Column) 추가하기

새로운 열을 추가할 때는 **cbind()** 함수를 사용합니다.

(cbind = Column Bind)

```r
matrix_a <- matrix(1:10,
                   nrow = 5,
                   byrow = TRUE)

matrix_a2 <- cbind(matrix_a,
                   1:5)

matrix_a2
```

결과

```text
     [,1] [,2] [,3]
[1,]    1    2    1
[2,]    3    4    2
[3,]    5    6    3
[4,]    7    8    4
[5,]    9   10    5
```

행의 개수는 그대로 유지되고,

열이 하나 증가한 것을 확인할 수 있습니다.

---

## 여러 개의 열 추가하기

행 개수만 같다면 여러 개의 열도 함께 붙일 수 있습니다.

```r
matrix_b <- matrix(13:24,
                   ncol = 3)

matrix_d <- cbind(matrix_b,
                  matrix_c)

dim(matrix_d)
```

결과

```r
[1] 4 6
```

---

# 행(Row) 추가하기

행을 추가할 때는 **rbind()** 함수를 사용합니다.

(Row Bind)

```r
matrix_c <- matrix(1:12,
                   ncol = 3)

new_row <- c(1,2,3)

matrix_c <- rbind(matrix_c,
                  new_row)

matrix_c
```

결과

```text
     [,1] [,2] [,3]
[1,]    1    5    9
[2,]    2    6   10
[3,]    3    7   11
[4,]    4    8   12
[5,]    1    2    3
```

---

# 행과 열 이름 붙이기

행렬에도 이름을 지정할 수 있습니다.

## 행 이름

```r
rownames(matrix_a) <- c(
  "Row1",
  "Row2",
  "Row3",
  "Row4",
  "Row5"
)
```

---

## 열 이름

```r
colnames(matrix_a) <- c(
  "Col1",
  "Col2"
)
```

출력 결과

```text
      Col1 Col2
Row1     1    2
Row2     3    4
Row3     5    6
Row4     7    8
Row5     9   10
```

행과 열 이름은 데이터 분석 결과를 해석할 때 매우 유용합니다.

---

# 행렬 데이터 선택하기 (Subsetting)

행렬은 대괄호 `[]`를 이용하여 원하는 부분만 선택할 수 있습니다.

기본 형태는 다음과 같습니다.

```r
행렬[행, 열]
```

---

## 특정 원소 선택

```r
matrix_c[1,2]
```

첫 번째 행, 두 번째 열의 값을 반환합니다.

---

## 여러 행과 열 선택

```r
matrix_c[1:3, 2:3]
```

1~3행,

2~3열만 선택합니다.

---

## 첫 번째 열 선택

```r
matrix_c[,1]
```

모든 행의 첫 번째 열을 반환합니다.

---

## 첫 번째 행 선택

```r
matrix_c[1,]
```

모든 열의 첫 번째 행을 반환합니다.

---

## 첫 번째 행 제외

```r
matrix_c[-1,]
```

첫 번째 행을 제외한 나머지 데이터를 반환합니다.

---

## 첫 번째 열 제외

```r
matrix_c[,-1]
```

첫 번째 열을 제외한 나머지를 반환합니다.

---

# 정리

- `matrix()` 함수로 행렬을 생성한다.
- `byrow` 옵션으로 데이터 입력 방향을 지정할 수 있다.
- `dim()`, `nrow()`, `ncol()`로 행렬 크기를 확인한다.
- `cbind()`는 열을 추가하고 `rbind()`는 행을 추가한다.
- `rownames()`와 `colnames()`로 행과 열 이름을 지정할 수 있다.
- `[]`를 이용하여 원하는 행과 열을 자유롭게 선택할 수 있다.

행렬은 이후 배우게 될 **데이터 프레임(Data Frame)** 의 기반이 되는 자료구조입니다.
