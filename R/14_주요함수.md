# 함수(Function)

## 함수란?

함수(Function)는 **특정 작업을 수행하기 위해 여러 명령어를 하나로 묶어놓은 코드 블록**이다.

프로그램을 작성하다 보면 동일한 코드를 여러 번 작성해야 하는 경우가 많다. 이때 함수를 사용하면 **코드의 재사용성을 높이고 유지보수를 쉽게 할 수 있다.**

R은 수백 개의 내장 함수(Built-in Function)를 제공하며, 필요하면 사용자가 직접 함수를 만들어 사용할 수도 있다.

함수는 일반적으로 다음과 같은 구조를 가진다.

- 입력(Input)
- 처리(Process)
- 결과(Output)

즉,

> **입력을 받아 특정 작업을 수행한 뒤 결과를 반환하는 것이 함수의 역할이다.**

---

# 함수의 기본 구조

함수의 기본 문법은 다음과 같다.

```r
function(argument){

  # 실행 코드

}
```

예를 들어 두 숫자를 더하는 함수는 다음과 같이 작성할 수 있다.

```r
add <- function(x, y){
  x + y
}

add(10, 20)
```

결과

```
[1] 30
```

---

# R의 내장 함수(Built-in Function)

R에는 이미 다양한 함수들이 준비되어 있다.

예를 들어

- mean()
- sum()
- max()
- min()
- sort()
- order()
- cbind()
- rbind()

등은 모두 R에서 기본 제공하는 함수이다.

```r
x <- c(10, 20, 30, 40)

sum(x)
mean(x)
max(x)
min(x)
```

결과

```
[1] 100
[1] 25
[1] 40
[1] 10
```

---

# 함수의 인수(Argument)

함수는 필요한 데이터를 **인수(Argument)** 로 전달받는다.

예를 들어 mean() 함수는 평균을 계산할 벡터를 전달받는다.

```r
score <- c(80, 90, 70)

mean(score)
```

결과

```
[1] 80
```

여러 개의 인수를 받을 수도 있다.

```r
round(3.141592, digits = 2)
```

결과

```
[1] 3.14
```

---

# 함수의 반환값(Return Value)

R의 대부분 함수는 계산 결과를 반환한다.

```r
result <- sqrt(49)

result
```

결과

```
[1] 7
```

반환된 값은 다른 변수에 저장하거나 이후 계산에 사용할 수 있다.

```r
x <- sqrt(100)

x + 20
```

결과

```
[1] 30
```

# 자주 사용하는 일반 함수

데이터 분석에서는 다양한 일반 함수들을 자주 사용한다.

| 함수 | 설명 |
|------|------|
| length() | 데이터 개수 확인 |
| sort() | 정렬 |
| order() | 정렬 순서 반환 |
| unique() | 중복 제거 |
| head() | 앞부분 출력 |
| tail() | 뒷부분 출력 |
| dim() | 행과 열 개수 확인 |
| nrow() | 행 개수 |
| ncol() | 열 개수 |

---

## 예제 1. length()

```r
fruit <- c("Apple", "Banana", "Orange", "Peach")

length(fruit)
```

결과

```
[1] 4
```

---

## 예제 2. sort()

```r
score <- c(90, 70, 80, 100)

sort(score)
```

결과

```
[1] 70 80 90 100
```

내림차순은 다음과 같이 작성한다.

```r
sort(score, decreasing = TRUE)
```

결과

```
[1] 100 90 80 70
```

---

## 예제 3. unique()

```r
city <- c("Seoul", "Busan", "Seoul", "Daegu")

unique(city)
```

결과

```
[1] "Seoul" "Busan" "Daegu"
```

---

## 예제 4. head()

```r
head(mtcars)
```

결과

```
앞의 6개 행 출력
```

---

## 예제 5. tail()

```r
tail(mtcars)
```

결과

```
마지막 6개 행 출력
```

# 수학 함수(Math Functions)

R은 다양한 수학 함수를 기본 제공한다.

| 함수 | 설명 |
|------|------|
| abs() | 절대값 |
| sqrt() | 제곱근 |
| log() | 로그 |
| exp() | 지수 |
| factorial() | 팩토리얼 |

---

## 예제 1. 절대값

```r
abs(-30)
```

결과

```
[1] 30
```

---

## 예제 2. 제곱근

```r
sqrt(64)
```

결과

```
[1] 8
```

---

## 예제 3. 로그

```r
log(100)
```

결과

```
[1] 4.60517
```

기본적으로 자연로그를 계산한다.

밑을 지정할 수도 있다.

```r
log(100, base = 10)
```

결과

```
[1] 2
```

---

## 예제 4. 지수 함수

```r
exp(2)
```

결과

```
[1] 7.389056
```

---

## 예제 5. 팩토리얼

```r
factorial(5)
```

결과

```
[1] 120
```

# 통계 함수(Statistical Functions)

R은 통계 분석을 위한 함수도 매우 풍부하다.

| 함수 | 설명 |
|------|------|
| mean() | 평균 |
| median() | 중앙값 |
| var() | 분산 |
| sd() | 표준편차 |
| summary() | 요약 통계 |
| quantile() | 분위수 |

---

## 예제 1. 평균

```r
score <- c(70, 80, 90, 100)

mean(score)
```

결과

```
[1] 85
```

---

## 예제 2. 중앙값

```r
median(score)
```

결과

```
[1] 85
```

---

## 예제 3. 표준편차

```r
sd(score)
```

결과

```
[1] 12.90994
```

---

## 예제 4. 분위수

```r
quantile(score)
```

결과

```
0%   25%   50%   75% 100%
70   77.5  85   92.5 100
```

---

## 예제 5. 요약 통계

```r
summary(score)
```

결과

```
Min.
1st Qu.
Median
Mean
3rd Qu.
Max.
```

데이터의 전반적인 특성을 빠르게 확인할 때 가장 많이 사용하는 함수 중 하나이다.

---

# 실무에서 많이 사용하는 함수

데이터 분석에서는 다음 함수들을 매우 자주 사용한다.

- mean()
- sum()
- nrow()
- ncol()
- dim()
- head()
- tail()
- unique()
- sort()
- order()
- summary()
- table()

특히 `summary()`와 `head()`는 데이터를 처음 확인할 때 거의 항상 사용하는 함수이다.

---

# 정리

- 함수는 특정 작업을 수행하는 코드 블록이다.
- 함수는 인수를 입력받아 결과를 반환한다.
- R에는 수백 개의 내장 함수가 존재한다.
- 데이터 분석에서는 일반 함수, 수학 함수, 통계 함수를 가장 많이 사용한다.
- 자주 사용하는 함수만 익혀도 대부분의 데이터 탐색 작업을 빠르게 수행할 수 있다.
