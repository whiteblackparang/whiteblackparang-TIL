# 데이터 불러오기

## 데이터 불러오기란?

데이터 분석의 첫 번째 단계는 **데이터를 R 환경으로 가져오는 것**이다.

실무에서는 데이터를 직접 입력하기보다 다음과 같은 파일을 불러와 분석하는 경우가 대부분이다.

- CSV
- Excel(xlsx, xls)
- TXT
- SPSS(.sav)
- Stata(.dta)
- SAS(.sas7bdat)

R은 다양한 패키지를 이용하여 거의 모든 형태의 데이터를 읽을 수 있다.

---

# 자주 사용하는 데이터 파일

| 파일 형식 | 확장자 | 사용 분야 |
|-----------|---------|-----------|
| CSV | .csv | 가장 많이 사용 |
| Excel | .xlsx, .xls | 기업 업무 |
| TXT | .txt | 로그 데이터 |
| SPSS | .sav | 통계 분석 |
| Stata | .dta | 경제학, 사회과학 |
| SAS | .sas7bdat | 의료, 금융 |

CSV와 Excel은 거의 모든 데이터 분석 프로젝트에서 사용된다.

---

# 현재 작업 폴더 확인하기

파일을 읽기 전에 현재 작업 폴더를 확인하는 것이 좋다.

```r
getwd()
```

### 결과(예시)

```text
[1] "C:/Users/User/Documents/R"
```

현재 위치를 기준으로 파일을 찾는다.

---

# 작업 폴더 변경하기

필요하다면 작업 폴더를 변경할 수 있다.

```r
setwd("C:/Data")
```

이후부터는 해당 폴더를 기준으로 파일을 읽는다.

---

# CSV 파일 읽기

가장 많이 사용하는 함수는 `read.csv()`이다.

## 예제 1

```r
df <- read.csv("sales.csv")

head(df)
```

### 결과(예시)

```text
  Product Sales
1       A   120
2       B   150
3       C   200
```

---

## 문자열을 문자형으로 읽기

최근 R에서는 기본값이 `stringsAsFactors = FALSE`이지만, 버전에 따라 명시하는 것이 좋다.

```r
df <- read.csv(
  "sales.csv",
  stringsAsFactors = FALSE
)
```

---

## 한글 파일 읽기

윈도우에서 저장한 CSV는 인코딩 문제가 발생할 수 있다.

```r
df <- read.csv(
  "korea.csv",
  fileEncoding = "CP949"
)
```

또는

```r
df <- read.csv(
  "korea.csv",
  fileEncoding = "UTF-8"
)
```

파일의 저장 방식에 따라 선택한다.

# readr 패키지 사용하기

`readr` 패키지는 기본 함수보다 빠르게 데이터를 읽을 수 있다.

```r
library(readr)

df <- read_csv("sales.csv")
```

### 장점

- 빠른 속도
- 큰 데이터 처리에 유리
- 데이터 타입 자동 인식

---

# Excel 파일 읽기

Excel 파일은 `readxl` 패키지를 많이 사용한다.

## 패키지 설치

```r

```

---

## 패키지 불러오기

```r
library(readxl)
```

---

## 예제 2

```r
sales <- read_excel("sales.xlsx")

```

---

## 시트 지정하기

```r
sales <- read_excel(
  "sales.xlsx",
  sheet = "2026"
```

```r
sales <- read_excel(
  "sales.xlsx",
```

```r

  "sales.xlsx",
```

# TXT 파일 읽기

구분자가 탭(Tab)인 경우

```r

df <- read.table("data.txt")
```

# 데이터 구조 확인하기

파일을 불러온 후 가장 먼저 확인해야 하는 함수들이다.

## head()

```r
head(df)
```

처음 6개 행을 출력한다.

---

## tail()

```r
tail(df)
```

마지막 6개 행을 출력한다.

---

## str()

```r
str(df)
```

변수의 자료형과 구조를 확인한다.

---

## summary()

```r
summary(df)
```

기초 통계량을 확인한다.

---

## dim()

```r
dim(df)
```

행과 열의 개수를 확인한다.

---

## names()

```r
names(df)
```

열 이름을 확인한다.

# 실무에서 가장 많이 사용하는 함수

| 함수 | 용도 |
|------|------|
| read.csv() | CSV 파일 읽기 |
| read_csv() | 대용량 CSV 읽기 |
| read_excel() | Excel 파일 읽기 |
| read_sav() | SPSS 파일 읽기 |
| read_dta() | Stata 파일 읽기 |
| read_sas() | SAS 파일 읽기 |
| head() | 데이터 미리 보기 |
| str() | 구조 확인 |
| summary() | 기초 통계 확인 |

---

# 정리
- `read.csv()`는 CSV 파일을 읽는 가장 기본적인 함수이다.
- Excel 파일은 `readxl` 패키지의 `read_excel()`을 사용한다.
- SPSS, Stata, SAS 파일은 `haven` 패키지로 읽을 수 있다.
- 데이터를 불러온 후에는 `head()`, `str()`, `summary()` 등을 통해 구조를 확인하는 것이 중요하다.
- 실무에서는 CSV와 Excel 파일을 가장 많이 사용하며, 데이터 구조를 먼저 점검하는 습관이 필요하다.
