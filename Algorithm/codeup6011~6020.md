## 6011 : 실수 1개 입력받아 변환하여 출력하기
- **문제**  변수에 실수값을 저장한 후 변수에 저장되어 있는 값을 그대로 출력한다
- **코드**
```python
f = input()
f = float(f)
print(f)
```

## 6012 : 정수 2개 입력받아 그대로 출력하기
- **문제** 줄을 바꿔 정수(integer) 2개를 입력받아 줄을 바꿔 출력한다
- **코드**
```python
a = input()
b = input()
a = int(a)
b = int(b)
print(a)
print(b)
```

## 6013 : 문자 2개 입력받아 순서 바꿔 출력하기
- **문제** 순서를 바꿔 한 줄에 한 문자씩 출력한다.
- **코드**
```python
a = input()
b = input()
c = a
a = b
b = c
print(a)
print(b)
```

## 6014 : 실수 1개 입력받아 3번 출력하기
- **문제** 입력받은 실수를 줄을 바꿔 3번 출력한다.
- **코드**
```python
a = input()
a = float(a)
print(a)
print(a)
print(a)
```

## 6015 : 정수 2개 입력받아 그대로 출력하기
- **문제** 입력된 두 정수를 줄을 바꿔 출력한다.
- **코드** 
```python
a, b = input().split()
a = int(a)
b = int(b)
print(a)
print(b)
```

## 6016 : 문자 2개 입력받아 순서 바꿔 출력하기
- **문제** 2개의 문자를 순서를 바꿔 한 줄로 출력한다.
- **코드** 
```python
c1, c2 = input().split()
print(c2, c1)
```
## 6017 : 문장 1개 입력받아 3번 출력하기
- **문제** 공백을 두고 3번 출력한다.
- **코드** 
```python
s = input()
print(s, s, s)
```

## 6018 : 시간 입력받아 그대로 출력하기
- **문제** 24시간 시:분 형식으로 시간이 입력될 때, 그대로 출력한다
- **코드**
```python
a, b = input().split(':')
print(a, b, sep=':')
```
## 6019 : 연월일 입력받아 순서 바꿔 출력하기
- **문제** "연도.월.일"을 입력받아 "일-월-연도" 순서로 바꿔 출력한다
- **코드**
```python
y, m, d = input().split('.')
print(d, m, y, sep='-')
```

## 6020 : 주민번호 입력받아 형태 바꿔 출력하기
- **문제** '-'를 제외한 주민번호 13자리를 모두 붙여 출력한다.
- **코드**
```python
a, b = input().split("-")
print(a, b, sep="")
```
